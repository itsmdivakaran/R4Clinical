18  Multivariate Methods in Clinical Research

18.1 Introduction

Multivariate statistical methods are essential tools in clinical research that allow analysts to examine multiple variables simultaneously, uncovering complex relationships and patterns in high-dimensional clinical data. While univariate methods analyze a single variable and bivariate methods examine relationships between two variables, multivariate approaches address real-world scenarios where numerous clinical measurements are collected on each patient.

18.1.1 Why Multivariate Methods Matter in Clinical Research

Clinical studies routinely collect dozens or hundreds of measurements per subject: laboratory values, vital signs, imaging metrics, genomic data, and patient-reported outcomes. Key motivations for multivariate analysis include:

  1. Dimensionality Reduction: Vital signs measured during clinic visits (systolic/diastolic BP, pulse, temperature, weight, height) are often correlated. PCA can reduce 5+ variables to 1-2 principal components while retaining 85-95% of total variance.

  2. Patient Stratification: Clustering methods identify subgroups of patients with similar clinical profiles, enabling precision medicine approaches.

  3. Confounding Control: Propensity score methods balance treatment groups on numerous confounders simultaneously, mimicking randomization in observational studies.

  4. Multiple Endpoint Analysis: MANOVA protects against inflated Type I error when testing whether a treatment affects multiple correlated outcomes simultaneously.

  5. Biomarker Discovery: Factor analysis and clustering reveal latent patterns in biomarker panels, such as liver function vs. renal function factors.

  6. Classification and Prediction: Discriminant analysis and logistic regression with multiple predictors enable patient risk stratification and treatment prediction.

18.1.2 Types of Multivariate Methods

This chapter covers six major categories:

  • Dimensionality Reduction: PCA, Factor Analysis — compress high-dimensional data while preserving information
  • Clustering: K-means, Hierarchical clustering — group similar observations without predefined classes
  • Multivariate Inference: MANOVA — test multiple correlated outcomes simultaneously
  • Causal Inference: Propensity score analysis — balance confounders in observational studies
  • Classification: Linear Discriminant Analysis (LDA) — predict class membership from multiple features
  • Longitudinal Analysis: Growth curves, repeated measures — analyze changes over time in multiple variables

This chapter progresses from theoretical foundations to practical clinical applications using the pharmaversesdtm dataset. All code is fully executable using base R and tidyverse packages, requiring only libraries commonly available in clinical research environments.

18.2 1. Principal Component Analysis (PCA)

18.2.1 Theory and Foundations

Principal Component Analysis is an unsupervised dimensionality reduction technique that transforms correlated variables into a smaller set of uncorrelated principal components (PCs).

Key Concepts:

  • Covariance Matrix: For p variables, the p × p covariance matrix Σ captures pairwise correlations.
  • Eigendecomposition: Σ = VΛV^T, where V contains eigenvectors (loadings) and Λ is diagonal with eigenvalues (variances of PCs).
  • Principal Components: PC₁ is the direction of maximum variance; PC₂ is orthogonal to PC₁ with second-largest variance.
  • Variance Explained: λₖ / Σλⱼ gives the proportion of total variance explained by the k-th PC.
  • Loadings: Correlations between original variables and PCs show which variables drive each component.

Clinical Relevance: Vital signs (systolic BP, diastolic BP, pulse, weight, height) are inherently correlated. A 10-unit increase in systolic BP often accompanies increases in diastolic BP. PCA reduces redundancy while preserving information about blood pressure profiles and body habitus.

18.3 2. Factor Analysis

18.3.1 Theory and Foundations

Factor Analysis attempts to explain correlations among observed variables using a smaller number of unobserved latent factors. Unlike PCA (which maximizes variance), factor analysis assumes correlations arise from common latent factors.

Key Concepts:

  • Common Factors: k << p latent variables that drive correlations among p observed variables
  • Factor Loadings: Weight (correlation) between observed variable and latent factor
  • Communalities: Proportion of variance in each variable explained by common factors
  • Specific Variance: Variance unique to each variable (not explained by common factors)
  • Rotations: Varimax (uncorrelated factors), promax (correlated factors) improve interpretability

Clinical Example: Lab values often cluster into functional groups: liver function (ALT, AST, BILI), renal function (CREAT, K). Factor analysis reveals these latent physiological patterns.

18.4 3. Cluster Analysis

18.4.1 3A. K-Means Clustering

Theory and Foundations

K-means partitions n observations into k clusters by minimizing within-cluster sum of squared deviations from cluster centroids. The algorithm iterates between:

  1. Assignment: Assign each observation to nearest centroid
  2. Update: Recompute centroids as cluster means

Challenges: - Choosing k (addressed via elbow method, silhouette analysis) - Local optima (mitigated by multiple random starts) - Assumes spherical clusters of similar size

Choosing k: Elbow Method

# Manual silhouette calculation
silhouette_manual <- function(data, clusters) {
  n <- nrow(data)
  k <- length(unique(clusters))
  silhouette_scores <- numeric(n)

  for (i in 1:n) {
    current_cluster <- clusters[i]

    # Within-cluster distance: average distance to other points in same cluster
    within_cluster_points <- which(clusters == current_cluster)

    if (length(within_cluster_points) > 1) {
      a_i <- mean(dist(rbind(data[i, ], data[within_cluster_points, ]))[-1])
    } else {
      a_i <- 0
    }

    # Nearest-cluster distance: average distance to points in nearest other cluster
    b_i <- Inf
    for (j in setdiff(1:k, current_cluster)) {
      other_cluster_points <- which(clusters == j)
      if (length(other_cluster_points) > 0) {
        d_ij <- mean(dist(rbind(data[i, ], data[other_cluster_points, ]))[-1])
        b_i <- min(b_i, d_ij)
      }
    }

    # Silhouette coefficient: avoid division by zero
    denominator <- max(a_i, b_i)
    if (denominator > 0) {

    }
  }

18.5 4. Multivariate Analysis of Variance (MANOVA)

18.5.1 Theory and Foundations

MANOVA tests whether group means differ on multiple correlated dependent variables simultaneously. It controls family-wise Type I error better than multiple univariate ANOVAs.

Test Statistics:

  • Pillai’s Trace: Σ(λᵢ / (1 + λᵢ)); 0 to p, robust to violations
  • Wilks’ Lambda: Π(1 / (1 + λᵢ)); 0 to 1, most common
  • Hotelling-Lawley Trace: Σλᵢ; 0 to ∞
  • Roy’s Greatest Root: Max(λᵢ); 0 to ∞

Assumptions: Multivariate normality, homogeneity of covariance matrices (Levene’s test, Box’s M-test)

18.6 5. Propensity Score Analysis

18.6.1 Theory and Foundations

Propensity score is the conditional probability of treatment assignment given baseline covariates: P(T = 1 | X). In observational studies, propensity score methods balance treatment groups on confounders, mimicking randomization.

Key Methods:

  1. Matching: Pair treated and control subjects with similar propensity scores
  2. Stratification: Create subgroups based on PS quintiles
  3. Weighting: IPTW (inverse probability of treatment weighting)

Assumptions: - Unconfoundedness: No unmeasured confounders (conditional on X) - Overlap: 0 < P(T=1|X) < 1 for all X (common support) - Consistency: SUTVA (stable unit treatment value assumption)

  for (var in c("AGE", "SEX_numeric", "RACE_numeric")) {

    vals_control <- as.numeric(data[[var]][data$Treatment == 0])

    treated_mean_before <- mean(vals_treated, na.rm = TRUE)
    control_mean_before <- mean(vals_control, na.rm = TRUE)

    var_treated <- var(vals_treated, na.rm = TRUE)
    var_control <- var(vals_control, na.rm = TRUE)
    pooled_var <- (var_treated + var_control) / 2

    # Avoid division by zero if pooled variance is zero or NA
    if (isTRUE(!is.na(pooled_var) && pooled_var > 0)) {

                    data.frame(
                      Variable = var,
                      Before_Treated = round(treated_mean_before, 3),
                      Before_Control = round(control_mean_before, 3),
                      Before_SMD = round(smd_before, 3),
                      Status = "Before"
                    ))
  }

  return(results)
}

18.7 6. Linear Discriminant Analysis (LDA)

18.7.1 Theory and Foundations

Linear Discriminant Analysis finds linear combinations of features that best separate predefined groups. It maximizes between-group variance relative to within-group variance.

Key Concepts:

  • Fisher’s Linear Discriminant: Direction w that maximizes (μ₁ - μ₂)² / (σ₁² + σ₂²)
  • Multivariate Extension: Multiple discriminant functions from covariance matrices
  • Classification: Assign observation to group with highest posterior probability

Clinical Use: Classify patients into disease stages or treatment response categories based on biomarker profiles.

18.9 8. Multiple Testing in Multivariate Settings

18.9.1 Background: The Multiple Comparisons Problem

When performing k statistical tests independently, the family-wise error rate (FWER) inflates:

FWER = 1 - (1 - α)^k

For 5 tests at α = 0.05: FWER = 1 - (0.95)^5 ≈ 0.226 (22.6% chance of any false positive!)

  7. Multiple Testing: Account for inflated false positive rate when testing many hypotheses. Use Bonferroni for strict FWER control or FDR (Benjamini-Hochberg) for exploratory studies.

# Multivariate Methods in Clinical Research


## Introduction

Multivariate statistical methods are essential tools in clinical research that allow analysts to examine multiple variables simultaneously, uncovering complex relationships and patterns in high-dimensional clinical data. While univariate methods analyze a single variable and bivariate methods examine relationships between two variables, multivariate approaches address real-world scenarios where numerous clinical measurements are collected on each patient.

### Why Multivariate Methods Matter in Clinical Research

Clinical studies routinely collect dozens or hundreds of measurements per subject: laboratory values, vital signs, imaging metrics, genomic data, and patient-reported outcomes. Key motivations for multivariate analysis include:

1. **Dimensionality Reduction**: Vital signs measured during clinic visits (systolic/diastolic BP, pulse, temperature, weight, height) are often correlated. PCA can reduce 5+ variables to 1-2 principal components while retaining 85-95% of total variance.

2. **Patient Stratification**: Clustering methods identify subgroups of patients with similar clinical profiles, enabling precision medicine approaches.

3. **Confounding Control**: Propensity score methods balance treatment groups on numerous confounders simultaneously, mimicking randomization in observational studies.

4. **Multiple Endpoint Analysis**: MANOVA protects against inflated Type I error when testing whether a treatment affects multiple correlated outcomes simultaneously.

5. **Biomarker Discovery**: Factor analysis and clustering reveal latent patterns in biomarker panels, such as liver function vs. renal function factors.

6. **Classification and Prediction**: Discriminant analysis and logistic regression with multiple predictors enable patient risk stratification and treatment prediction.

### Types of Multivariate Methods

This chapter covers six major categories:

- **Dimensionality Reduction**: PCA, Factor Analysis — compress high-dimensional data while preserving information
- **Clustering**: K-means, Hierarchical clustering — group similar observations without predefined classes
- **Multivariate Inference**: MANOVA — test multiple correlated outcomes simultaneously
- **Causal Inference**: Propensity score analysis — balance confounders in observational studies
- **Classification**: Linear Discriminant Analysis (LDA) — predict class membership from multiple features
- **Longitudinal Analysis**: Growth curves, repeated measures — analyze changes over time in multiple variables

## 1. Principal Component Analysis (PCA)

### Theory and Foundations

Principal Component Analysis is an unsupervised dimensionality reduction technique that transforms correlated variables into a smaller set of uncorrelated principal components (PCs).

**Key Concepts:**

- **Covariance Matrix**: For p variables, the p × p covariance matrix Σ captures pairwise correlations.
- **Eigendecomposition**: Σ = VΛV^T, where V contains eigenvectors (loadings) and Λ is diagonal with eigenvalues (variances of PCs).
- **Principal Components**: PC₁ is the direction of maximum variance; PC₂ is orthogonal to PC₁ with second-largest variance.
- **Variance Explained**: λₖ / Σλⱼ gives the proportion of total variance explained by the k-th PC.
- **Loadings**: Correlations between original variables and PCs show which variables drive each component.

**Clinical Relevance:** Vital signs (systolic BP, diastolic BP, pulse, weight, height) are inherently correlated. A 10-unit increase in systolic BP often accompanies increases in diastolic BP. PCA reduces redundancy while preserving information about blood pressure profiles and body habitus.

## 2. Factor Analysis

### Theory and Foundations

Factor Analysis attempts to explain correlations among observed variables using a smaller number of unobserved latent factors. Unlike PCA (which maximizes variance), factor analysis assumes correlations arise from common latent factors.

**Key Concepts:**

- **Common Factors**: k << p latent variables that drive correlations among p observed variables
- **Factor Loadings**: Weight (correlation) between observed variable and latent factor
- **Communalities**: Proportion of variance in each variable explained by common factors
- **Specific Variance**: Variance unique to each variable (not explained by common factors)
- **Rotations**: Varimax (uncorrelated factors), promax (correlated factors) improve interpretability

**Clinical Example:** Lab values often cluster into functional groups: liver function (ALT, AST, BILI), renal function (CREAT, K). Factor analysis reveals these latent physiological patterns.

#### Theory and Foundations

K-means partitions n observations into k clusters by minimizing within-cluster sum of squared deviations from cluster centroids. The algorithm iterates between:

1. **Assignment**: Assign each observation to nearest centroid
2. **Update**: Recompute centroids as cluster means

**Challenges:**
- Choosing k (addressed via elbow method, silhouette analysis)
- Local optima (mitigated by multiple random starts)
- Assumes spherical clusters of similar size

```{r}
# Manual silhouette calculation
silhouette_manual <- function(data, clusters) {
  n <- nrow(data)
  k <- length(unique(clusters))
  silhouette_scores <- numeric(n)

  for (i in 1:n) {
    current_cluster <- clusters[i]

    # Within-cluster distance: average distance to other points in same cluster
    within_cluster_points <- which(clusters == current_cluster)

    if (length(within_cluster_points) > 1) {
      a_i <- mean(dist(rbind(data[i, ], data[within_cluster_points, ]))[-1])
    } else {
      a_i <- 0
    }

    # Nearest-cluster distance: average distance to points in nearest other cluster
    b_i <- Inf
    for (j in setdiff(1:k, current_cluster)) {
      other_cluster_points <- which(clusters == j)
      if (length(other_cluster_points) > 0) {
        d_ij <- mean(dist(rbind(data[i, ], data[other_cluster_points, ]))[-1])
        b_i <- min(b_i, d_ij)
      }
    }

    # Silhouette coefficient: avoid division by zero
    denominator <- max(a_i, b_i)
    if (denominator > 0) {
      silhouette_scores[i] <- (b_i - a_i) / denominator
    } else {
      silhouette_scores[i] <- 0
    }
  }

  return(silhouette_scores)
}
```

## 4. Multivariate Analysis of Variance (MANOVA)

### Theory and Foundations

MANOVA tests whether group means differ on multiple correlated dependent variables simultaneously. It controls family-wise Type I error better than multiple univariate ANOVAs.

**Test Statistics:**

- **Pillai's Trace**: Σ(λᵢ / (1 + λᵢ)); 0 to p, robust to violations
- **Wilks' Lambda**: Π(1 / (1 + λᵢ)); 0 to 1, most common
- **Hotelling-Lawley Trace**: Σλᵢ; 0 to ∞
- **Roy's Greatest Root**: Max(λᵢ); 0 to ∞

**Assumptions:** Multivariate normality, homogeneity of covariance matrices (Levene's test, Box's M-test)

## 5. Propensity Score Analysis

### Theory and Foundations

Propensity score is the conditional probability of treatment assignment given baseline covariates: P(T = 1 | X). In observational studies, propensity score methods balance treatment groups on confounders, mimicking randomization.

**Key Methods:**

1. **Matching**: Pair treated and control subjects with similar propensity scores
2. **Stratification**: Create subgroups based on PS quintiles
3. **Weighting**: IPTW (inverse probability of treatment weighting)

**Assumptions:**
- **Unconfoundedness**: No unmeasured confounders (conditional on X)
- **Overlap**: 0 < P(T=1|X) < 1 for all X (common support)
- **Consistency**: SUTVA (stable unit treatment value assumption)

```{r}
balance_comparison <- function(data, treated_idx) {
  results <- data.frame()

  for (var in c("AGE", "SEX_numeric", "RACE_numeric")) {
    # Use [[ ]] to extract a plain numeric vector (not a tibble column)
    vals_treated <- as.numeric(data[[var]][data$Treatment == 1])
    vals_control <- as.numeric(data[[var]][data$Treatment == 0])

    treated_mean_before <- mean(vals_treated, na.rm = TRUE)
    control_mean_before <- mean(vals_control, na.rm = TRUE)

    var_treated <- var(vals_treated, na.rm = TRUE)
    var_control <- var(vals_control, na.rm = TRUE)
    pooled_var <- (var_treated + var_control) / 2

    # Avoid division by zero if pooled variance is zero or NA
    if (isTRUE(!is.na(pooled_var) && pooled_var > 0)) {
      smd_before <- (treated_mean_before - control_mean_before) / sqrt(pooled_var)
    } else {
      smd_before <- NA
    }

                    data.frame(
                      Variable = var,
                      Before_Treated = round(treated_mean_before, 3),
                      Before_Control = round(control_mean_before, 3),
                      Before_SMD = round(smd_before, 3),
                      Status = "Before"
                    ))
  }

  return(results)
}
```

## 6. Linear Discriminant Analysis (LDA)

### Theory and Foundations

Linear Discriminant Analysis finds linear combinations of features that best separate predefined groups. It maximizes between-group variance relative to within-group variance.

**Key Concepts:**

- **Fisher's Linear Discriminant**: Direction w that maximizes (μ₁ - μ₂)² / (σ₁² + σ₂²)
- **Multivariate Extension**: Multiple discriminant functions from covariance matrices
- **Classification**: Assign observation to group with highest posterior probability

**Clinical Use:** Classify patients into disease stages or treatment response categories based on biomarker profiles.

## 8. Multiple Testing in Multivariate Settings

### Background: The Multiple Comparisons Problem

When performing k statistical tests independently, the family-wise error rate (FWER) inflates:

FWER = 1 - (1 - α)^k

For 5 tests at α = 0.05: FWER = 1 - (0.95)^5 ≈ 0.226 (22.6% chance of any false positive!)

### Multiple Testing Corrections


```{r}
# Simulate p-values from a study with 20 independent tests
# Assume true null for all tests (no real effects)

n_tests <- 20
n_simulations <- 1000

# Generate random p-values under null hypothesis
pvalues_matrix <- matrix(runif(n_tests * n_simulations),
                        nrow = n_simulations,
                        ncol = n_tests)

# Apply different correction methods
uncorrected_reject <- apply(pvalues_matrix < 0.05, 1, sum)
bonferroni_reject <- apply(pvalues_matrix < 0.05/n_tests, 1, sum)
holm_rejected <- numeric(n_simulations)
fdr_rejected <- numeric(n_simulations)

for (i in 1:n_simulations) {
  # Holm-Bonferroni
  holm_corrected <- p.adjust(pvalues_matrix[i, ], method = "holm")
  holm_rejected[i] <- sum(holm_corrected < 0.05)

  # Benjamini-Hochberg (FDR)
  fdr_corrected <- p.adjust(pvalues_matrix[i, ], method = "BH")
  fdr_rejected[i] <- sum(fdr_corrected < 0.05)
}

# Summary statistics
cat("Multiple Testing Simulation Results\n")
```

## 9. Summary: Choosing the Right Multivariate Method

| **Problem** | **Method** | **Use Case** | **Key Assumption** |
|---|---|---|---|
| Reduce 10 correlated variables to 3 principal components | PCA | Exploratory data analysis, data compression | None (unsupervised) |
| Identify latent factors explaining correlations | Factor Analysis | Biomarker discovery, construct validation | Multivariate normality |
| Group subjects into homogeneous clusters | K-Means or Hierarchical | Patient stratification, phenotyping | Spherical clusters (K-means) |
| Test treatment effect on 3 correlated outcomes | MANOVA | Multiple related endpoints | Multivariate normality |
| Balance confounders in observational study | Propensity Scoring | Causal inference without RCT | Unconfoundedness, overlap |
| Classify subjects into predefined groups | LDA | Risk stratification, diagnosis prediction | Multivariate normality, equal covariances |
| Model outcome trends over multiple visits | Growth Curves / MANOVA repeated measures | Longitudinal analysis | Appropriate covariance structure |
| Control false positive rate across 20 biomarkers | Multiple testing correction (Holm, FDR) | Biomarker discovery, panel testing | Depends on method |

---

## 10. Key Takeaways

1. **Dimensionality Reduction** (PCA, Factor Analysis): Simplify high-dimensional data while preserving structure. PCA is unsupervised and variance-focused; FA explains correlations via latent factors.

2. **Clustering** (K-Means, Hierarchical): Partition subjects into similar groups. Use elbow method or silhouette analysis to choose k. Hierarchical clustering provides a dendrogram for visual exploration.

3. **Multivariate Inference** (MANOVA): Test multiple correlated endpoints simultaneously, controlling family-wise Type I error better than separate ANOVAs.

4. **Causal Inference** (Propensity Scores): Balance treatment groups on confounders in observational studies. Check overlap assumption and assess covariate balance before/after matching.

5. **Classification** (LDA): Assign new observations to predefined classes based on multiple features. Provides interpretable linear discriminant scores.

6. **Longitudinal Analysis** (Growth Curves): Model trends over time by treatment group. Use polynomial models or more complex mixed-effects approaches.

7. **Multiple Testing**: Account for inflated false positive rate when testing many hypotheses. Use Bonferroni for strict FWER control or FDR (Benjamini-Hochberg) for exploratory studies.
