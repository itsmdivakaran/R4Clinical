35  Oncology Trial Case Study

This chapter presents a comprehensive, real-world case study of a Phase III oncology trial analyzing efficacy and safety data. We demonstrate survival analysis, response evaluation, subgroup analyses, and safety reporting using clinical datasets and rigorous biostatistical methods aligned with oncology regulatory standards.

35.1 24.1 Background: A Phase III Oncology Trial

35.1.1 The ONCO-301 Trial Design

Imagine we are analyzing data from ONCO-301, a pivotal Phase III randomised controlled trial evaluating a novel targeted therapy in metastatic colorectal cancer (mCRC). The trial compares:

  • Investigational arm: Vemotinib (a hypothetical BRAF inhibitor), 960 mg once daily
  • Control arm: FOLFOX (fluorouracil, leucovorin, oxaliplatin) standard chemotherapy

Trial characteristics: - 500 patients, 1:1 randomisation - Inclusion: histologically confirmed mCRC, ECOG PS 0–1, prior treatment naïve - Stratification: by site and BRAF mutation status - Primary endpoint: Overall Survival (OS) — time from randomisation to death from any cause - Secondary endpoints: Progression-Free Survival (PFS), Objective Response Rate (ORR)

35.1.2 Clinical Endpoints Explained

NotePrimary and Secondary Endpoints in Oncology

Overall Survival (OS): Time from randomisation to death from any cause. The gold standard efficacy endpoint, directly measuring patient benefit. Regulatory agencies prioritise OS in their approval decisions.

Progression-Free Survival (PFS): Time from randomisation to documented disease progression (per RECIST 1.1) or death from any cause, whichever occurs first. A surrogate endpoint reflecting treatment effect on tumour control.

Objective Response Rate (ORR): Percentage of patients achieving Complete Response (CR) or Partial Response (PR) per RECIST 1.1 criteria. An intermediate endpoint assessing tumour shrinkage.

35.1.3 RECIST 1.1 Response Categories

RECIST 1.1 defines tumour response based on change in target lesion diameter from baseline:

  • Complete Response (CR): Disappearance of all target lesions. ≥30% decrease from baseline.
  • Partial Response (PR): ≥30% decrease in sum of target lesion diameters.
  • Stable Disease (SD): Neither PR nor PD criteria met.
  • Progressive Disease (PD): ≥20% increase in sum of target lesion diameters from nadir.

The colon dataset contains 1,858 observations with two rows per subject (one for each event type). Key variables:

  • id: Subject identifier
  • time: Follow-up time in days from entry
  • status: Event indicator (1 = recurrence/progression, 2 = death)
  • etype: Event type (1 = recurrence, 2 = death)
  • rx: Treatment (Obs, Lev, Lev+5FU)
  • age: Age in years
  • sex: 1 = male, 0 = female
  • obstruct: Obstruction of colon (1 = yes, 0 = no)
  • perfor: Perforation (1 = yes, 0 = no)
  • adhere: Adherence (1 = yes, 0 = no)
  • nodes: Number of lymph nodes examined
  • node4: Number of positive lymph nodes (1 = 4+, 0 = <4)
  • differ: Grade of differentiation (1 = well, 2 = moderate, 3 = poor)
  • extent: Extent of spread (1 = confined, 2 = submucosa, 3 = muscle, 4 = beyond)
  • surg: Type of surgery (0 = short, 1 = long)

  filter(etype == 2) %>%
  select(id, time, status, rx, age, sex, obstruct, perfor, adhere,

  # Map treatments: "Obs" → Control, "Lev+5FU" → Vemotinib

recist_data <- colon_os_unique %>%
  select(subject_id, id, rx, age, node4) %>%
  mutate(
    # Simulate best response by treatment
    best_response = {
      resp    <- character(n())
      is_vemo <- rx == "Vemotinib"
      resp[is_vemo]  <- sample(c("CR","PR","SD","PD"), sum(is_vemo),  replace=TRUE,
                                prob=c(0.12, 0.28, 0.35, 0.25))
      resp[!is_vemo] <- sample(c("CR","PR","SD","PD"), sum(!is_vemo), replace=TRUE,
                                prob=c(0.05, 0.15, 0.40, 0.40))

      chg     <- numeric(n())

35.3 24.3 Overall Survival Analysis

35.3.1 Kaplan-Meier Estimator

The Kaplan-Meier (KM) estimator provides a non-parametric estimate of the survival function:

\[S(t) = \prod_{i: t_i \leq t} \left(1 - \frac{d_i}{n_i}\right)\]

where: - \(d_i\) = number of events at time \(t_i\) - \(n_i\) = number at risk at time \(t_i\)

35.3.2 Fitting the KM Estimator

# Fit Kaplan-Meier curves by treatment

\[H_0: S_{\text{Vemotinib}}(t) = S_{\text{Control}}(t) \text{ for all } t\]

35.4 Clinical Interpretation

The log-rank test result indicates whether Vemotinib provides a statistically significant survival benefit compared to FOLFOX. If p < 0.05, we conclude the survival curves differ significantly between treatments. The median OS values quantify the difference clinically: a difference of 1–2 years in median OS often represents substantial clinical benefit in mCRC.

35.5 24.4 Cox Proportional Hazards Model for Overall Survival

35.5.1 Fitting the Cox Model

The Cox proportional hazards model relates covariates to the hazard rate:

\[h(t | X) = h_0(t) \exp(\beta_1 X_1 + \beta_2 X_2 + \cdots + \beta_p X_p)\]

The hazard ratio for treatment is \(\exp(\beta_{\text{rx}})\), representing the relative risk of death.

# Fit Cox model with treatment and key prognostic factors
cox_os <- coxph(Surv(time_years, os_event) ~ rx + age + sex + obstruct + node4,
                data = colon_os)

## 24.1 Background: A Phase III Oncology Trial

### The ONCO-301 Trial Design

Imagine we are analyzing data from **ONCO-301**, a pivotal Phase III randomised controlled trial evaluating a novel targeted therapy in metastatic colorectal cancer (mCRC). The trial compares:

- **Investigational arm**: Vemotinib (a hypothetical BRAF inhibitor), 960 mg once daily
- **Control arm**: FOLFOX (fluorouracil, leucovorin, oxaliplatin) standard chemotherapy

**Trial characteristics**:
- 500 patients, 1:1 randomisation
- Inclusion: histologically confirmed mCRC, ECOG PS 0–1, prior treatment naïve
- Stratification: by site and BRAF mutation status
- Primary endpoint: **Overall Survival (OS)** — time from randomisation to death from any cause
- Secondary endpoints: **Progression-Free Survival (PFS)**, **Objective Response Rate (ORR)**

### Clinical Endpoints Explained

::: {.callout-note}
## Primary and Secondary Endpoints in Oncology

**Overall Survival (OS)**: Time from randomisation to death from any cause. The gold standard efficacy endpoint, directly measuring patient benefit. Regulatory agencies prioritise OS in their approval decisions.

**Progression-Free Survival (PFS)**: Time from randomisation to documented disease progression (per RECIST 1.1) or death from any cause, whichever occurs first. A surrogate endpoint reflecting treatment effect on tumour control.

**Objective Response Rate (ORR)**: Percentage of patients achieving Complete Response (CR) or Partial Response (PR) per RECIST 1.1 criteria. An intermediate endpoint assessing tumour shrinkage.
:::

### RECIST 1.1 Response Categories

RECIST 1.1 defines tumour response based on change in target lesion diameter from baseline:

- **Complete Response (CR)**: Disappearance of all target lesions. ≥30% decrease from baseline.
- **Partial Response (PR)**: ≥30% decrease in sum of target lesion diameters.
- **Stable Disease (SD)**: Neither PR nor PD criteria met.
- **Progressive Disease (PD)**: ≥20% increase in sum of target lesion diameters from nadir.

## 24.3 Overall Survival Analysis

### Kaplan-Meier Estimator

The Kaplan-Meier (KM) estimator provides a non-parametric estimate of the survival function:

$$S(t) = \prod_{i: t_i \leq t} \left(1 - \frac{d_i}{n_i}\right)$$

where:
- $d_i$ = number of events at time $t_i$
- $n_i$ = number at risk at time $t_i$

## 24.4 Cox Proportional Hazards Model for Overall Survival

### Fitting the Cox Model

The Cox proportional hazards model relates covariates to the hazard rate:

$$h(t | X) = h_0(t) \exp(\beta_1 X_1 + \beta_2 X_2 + \cdots + \beta_p X_p)$$

The hazard ratio for treatment is $\exp(\beta_{\text{rx}})$, representing the relative risk of death.

```{r}
#| label: cox-os-fit
#| eval: true

# Fit Cox model with treatment and key prognostic factors
cox_os <- coxph(Surv(time_years, os_event) ~ rx + age + sex + obstruct + node4,
                data = colon_os)

# Display model summary
summary(cox_os)
```

### Checking Proportional Hazards Assumption

The Cox model assumes proportional hazards: the hazard ratio between groups is constant over time. We test this using Schoenfeld residuals.

```{r}
#| label: cox-ph-test
#| eval: true

# Test proportional hazards assumption
ph_test <- cox.zph(cox_os)
print(ph_test)

# Interpretation: large p-values (>0.05) suggest the PH assumption holds
```

::: {.callout-warning}
The proportional hazards assumption is critical for Cox model validity. If any variable has p < 0.05 in the `cox.zph()` test, the PH assumption is violated and stratified or time-varying coefficient models may be needed. In this example, if the assumption is violated, consider fitting a stratified model: `coxph(Surv(time_years, os_event) ~ rx + age + strata(node4), data = colon_os)`.
:::
