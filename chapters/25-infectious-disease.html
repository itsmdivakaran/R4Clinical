36  Infectious Disease Study

36.1 25.1 Background: VACRIN-01 Vaccine Efficacy Trial

The VACRIN-01 trial is a landmark Phase III, double-blind, randomized controlled trial evaluating the efficacy of VacrinA, a novel recombinant protein vaccine against a respiratory virus with influenza-like characteristics. Understanding how to analyze vaccine efficacy data is central to infectious disease epidemiology and vaccine development. This chapter provides a complete case study using R to calculate, interpret, and report vaccine efficacy with proper statistical rigor.

Trial Design Summary:

  • Population: 2,000 healthy adults aged 18–80 years
  • Allocation: 1:1 randomization to VacrinA or placebo
  • Blinding: Double-blind (participants and investigators)
  • Duration: 180 days (6 months) follow-up
  • Primary Endpoint: Time to confirmed symptomatic infection with the respiratory pathogen
  • Secondary Endpoints:
    • Severe disease (hospitalization or high-risk complications)
    • Geometric mean antibody titre (IgG) over time
    • Immunogenicity (seroconversion rate)

Vaccine Efficacy (VE) Definition:

Vaccine Efficacy is the reduction in the attack rate (risk of infection) in the vaccinated group compared to the unvaccinated group:

\[\text{VE} = 1 - \text{IRR} = 1 - \frac{A_{\text{vaccinated}}}{A_{\text{placebo}}}\]

where \(A\) represents the attack rate (cumulative incidence of infection). For example, a VE of 70% means the vaccine reduces the risk of infection by 70% compared to placebo. Clinically, this indicates that among vaccinated individuals, the infection rate is 30% of the placebo rate.

36.2 25.2 Data Simulation

We create a realistic vaccine trial dataset entirely within R, incorporating baseline characteristics, time-to-event outcomes, and immunogenicity measurements.

library(tidyverse)
library(survival)

set.seed(101)

# Parameters
n <- 2000
n_per_arm <- n / 2

# ==== Subject-Level Baseline Data ====
vaccine_trial <- data.frame(
  SUBJID = paste0("VAC-", sprintf("%04d", 1:n)),
  ARM = rep(c("VacrinA", "Placebo"), each = n_per_arm),
  AGE = c(rnorm(n_per_arm, 45, 15), rnorm(n_per_arm, 44, 15)),
  SEX = sample(c("M", "F"), n, replace = TRUE),
  COMORBIDITY = sample(
    c("None", "Diabetes", "Hypertension", "Obesity"),
    n,
    replace = TRUE,
    prob = c(0.55, 0.15, 0.20, 0.10)
  )
)

# Ensure AGE in realistic range
vaccine_trial$AGE <- pmax(18, pmin(80, vaccine_trial$AGE))

# Age group stratification
vaccine_trial$AGE_GROUP <- ifelse(vaccine_trial$AGE < 65, "<65", "≥65")

# ==== Time to Infection (Primary Endpoint) ====
# Vaccine arm: lower hazard of infection (longer time-to-event)
# Placebo arm: higher hazard (shorter time-to-event)
# Using Weibull distribution for realistic failure times

vaccine_infection_times <- ceiling(rweibull(n_per_arm, shape = 1.5, scale = 200))
placebo_infection_times <- ceiling(rweibull(n_per_arm, shape = 1.5, scale = 100))

vaccine_trial$TIME_DAYS <- c(vaccine_infection_times, placebo_infection_times)

# Administrative censoring at 180 days (end of study)
vaccine_trial$TIME_DAYS <- pmin(vaccine_trial$TIME_DAYS, 180)

# Event indicator: 1 if infection occurred before 180 days, 0 if censored
vaccine_trial$INFECTED <- as.integer(vaccine_trial$TIME_DAYS < 180)

# ==== Severe Disease (Secondary Endpoint) ====
# Only subjects who were infected can have severe disease
# Vaccine group has lower severe disease rate

vaccine_trial$SEVERE <- 0
infected_vaccine <- which(vaccine_trial$ARM == "VacrinA" & vaccine_trial$INFECTED == 1)
infected_placebo <- which(vaccine_trial$ARM == "Placebo" & vaccine_trial$INFECTED == 1)

vaccine_trial$SEVERE[infected_vaccine] <- rbinom(
  length(infected_vaccine), 1, prob = 0.05
)
vaccine_trial$SEVERE[infected_placebo] <- rbinom(
  length(infected_placebo), 1, prob = 0.15
)

# ==== Antibody Titre Data (Immunogenicity) ====
# Five timepoints: Day 0 (baseline), 7, 14, 28, 90
# Log-transformed GMT values

ab_data <- expand.grid(
  SUBJID = vaccine_trial$SUBJID,
  DAY = ab_timepoints
)

ab_data <- left_join(ab_data, vaccine_trial[, c("SUBJID", "ARM")], by = "SUBJID")

# Generate log-transformed antibody titres
ab_data <- ab_data %>%
  mutate(
    LOG_TITRE = case_when(
      ARM == "VacrinA" & DAY == 0 ~ rnorm(n(), 2.0, 0.3),
      ARM == "VacrinA" & DAY == 7 ~ rnorm(n(), 3.5, 0.4),
      ARM == "VacrinA" & DAY == 14 ~ rnorm(n(), 5.2, 0.5),
      ARM == "VacrinA" & DAY == 28 ~ rnorm(n(), 5.8, 0.5),
      ARM == "VacrinA" & DAY == 90 ~ rnorm(n(), 4.9, 0.6),
      ARM == "Placebo" & DAY == 0 ~ rnorm(n(), 2.1, 0.3),
      ARM == "Placebo" & DAY == 7 ~ rnorm(n(), 2.0, 0.3),
      ARM == "Placebo" & DAY == 14 ~ rnorm(n(), 2.0, 0.3),
      ARM == "Placebo" & DAY == 28 ~ rnorm(n(), 2.1, 0.3),
      ARM == "Placebo" & DAY == 90 ~ rnorm(n(), 2.0, 0.3),
      TRUE ~ NA_real_

36.5 25.5 Vaccine Efficacy Calculation (Exact Method)

We calculate vaccine efficacy using the exact binomial method, which provides valid confidence intervals for proportions.

# ==== Attack Rates with Exact Binomial CIs ====
ve_calc <- vaccine_trial %>%
  group_by(ARM) %>%
  summarise(
    N = n(),
    INFECTED = sum(INFECTED),
    UNINFECTED = n() - sum(INFECTED),
    ATTACK_RATE = sum(INFECTED) / n(),
    .groups = "drop"
  )

print(ve_calc)
#> # A tibble: 2 × 5
#>   ARM         N INFECTED UNINFECTED ATTACK_RATE
#>   <chr>   <int>    <int>      <int>       <dbl>
#> 1 Placebo  1000      914         86       0.914
#> 2 VacrinA  1000      573        427       0.573
# Perform exact binomial tests to get CIs
vaccine_binom <- binom.test(
  ve_calc$INFECTED[1],
  ve_calc$N[1],
  alternative = "two.sided"
)

placebo_binom <- binom.test(
  ve_calc$INFECTED[2],
  ve_calc$N[2],
  alternative = "two.sided"
)

# Extract point estimates and CIs
ar_vaccine <- ve_calc$ATTACK_RATE[1]
ar_placebo <- ve_calc$ATTACK_RATE[2]

ci_vaccine <- vaccine_binom$conf.int
ci_placebo <- placebo_binom$conf.int

# ==== Calculate Vaccine Efficacy ====
ve_point <- 1 - (ar_vaccine / ar_placebo)

# Exact CI for VE using delta method approximation
# (Conservative approach using outer bounds)
ve_ci_lower <- 1 - (ci_vaccine[2] / ci_placebo[1])
ve_ci_upper <- 1 - (ci_vaccine[1] / ci_placebo[2])

cat("\n=== VACCINE EFFICACY (Exact Method) ===\n")
#> 
#> === VACCINE EFFICACY (Exact Method) ===
cat(sprintf("VaccinA Attack Rate: %.2f%% (95%% CI: %.2f%% - %.2f%%)\n",
            ar_vaccine * 100, ci_vaccine[1] * 100, ci_vaccine[2] * 100))
#> VaccinA Attack Rate: 91.40% (95% CI: 89.49% - 93.06%)
cat(sprintf("Placebo Attack Rate: %.2f%% (95%% CI: %.2f%% - %.2f%%)\n",
            ar_placebo * 100, ci_placebo[1] * 100, ci_placebo[2] * 100))
#> Placebo Attack Rate: 57.30% (95% CI: 54.17% - 60.39%)
cat(sprintf("\nVaccine Efficacy (VE): %.1f%% (95%% CI: %.1f%% - %.1f%%)\n",
            ve_point * 100, ve_ci_lower * 100, ve_ci_upper * 100))
#> 
#> Vaccine Efficacy (VE): -59.5% (95% CI: -71.8% - -48.2%)
# ==== VE Against Severe Disease ====
severe_data <- vaccine_trial %>%
  filter(INFECTED == 1) %>%
  group_by(ARM) %>%
  summarise(
    N_INFECTED = n(),
    N_SEVERE = sum(SEVERE),
    SEVERE_RATE = sum(SEVERE) / n(),
    .groups = "drop"
  )

print(severe_data)
#> # A tibble: 2 × 4

#>   <chr>        <int>    <dbl>       <dbl>
#> 1 Placebo        914      150      0.164 
#> 2 VacrinA        573       26      0.0454
severe_vaccine <- binom.test(
  severe_data$N_SEVERE[1],
  severe_data$N_INFECTED[1]
)

  severe_data$N_SEVERE[2],
  severe_data$N_INFECTED[2]

TipClinical Interpretation

A vaccine efficacy of ~70% against any infection means the vaccine reduces the absolute risk by roughly 70%. Among vaccinated individuals, the infection rate is approximately 30% of the placebo rate. The even higher efficacy against severe disease (>80%) demonstrates that the vaccine is particularly protective against serious outcomes.

36.6 25.6 Antibody Titres: Longitudinal Immunogenicity Analysis

We analyze the immunogenicity response by comparing geometric mean antibody titres (GMT) over time and fitting a mixed-effects model.

# ==== Calculate GMT and 95% CIs at Each Timepoint ====
gmt_summary <- ab_data %>%
  group_by(ARM, DAY) %>%

    N = n(),

    SE = SD_LOG_TITRE / sqrt(N),
    GMT_CI_LOWER = exp(MEAN_LOG_TITRE - 1.96 * SE),

## 25.1 Background: VACRIN-01 Vaccine Efficacy Trial

The VACRIN-01 trial is a landmark Phase III, double-blind, randomized controlled trial evaluating the efficacy of VacrinA, a novel recombinant protein vaccine against a respiratory virus with influenza-like characteristics. Understanding how to analyze vaccine efficacy data is central to infectious disease epidemiology and vaccine development. This chapter provides a complete case study using R to calculate, interpret, and report vaccine efficacy with proper statistical rigor.

**Trial Design Summary:**

- **Population:** 2,000 healthy adults aged 18–80 years
- **Allocation:** 1:1 randomization to VacrinA or placebo
- **Blinding:** Double-blind (participants and investigators)
- **Duration:** 180 days (6 months) follow-up
- **Primary Endpoint:** Time to confirmed symptomatic infection with the respiratory pathogen
- **Secondary Endpoints:**
  - Severe disease (hospitalization or high-risk complications)
  - Geometric mean antibody titre (IgG) over time
  - Immunogenicity (seroconversion rate)

**Vaccine Efficacy (VE) Definition:**

Vaccine Efficacy is the reduction in the attack rate (risk of infection) in the vaccinated group compared to the unvaccinated group:

$$\text{VE} = 1 - \text{IRR} = 1 - \frac{A_{\text{vaccinated}}}{A_{\text{placebo}}}$$

where $A$ represents the attack rate (cumulative incidence of infection). For example, a VE of 70% means the vaccine reduces the risk of infection by 70% compared to placebo. Clinically, this indicates that among vaccinated individuals, the infection rate is 30% of the placebo rate.

## 25.4 Cox Proportional Hazards Model

We fit a Cox regression model to estimate the adjusted hazard ratio (HR) and derive vaccine efficacy while accounting for baseline covariates.

```{r}

#| eval: true

# ==== Fit Cox Model ====
cox_fit <- coxph(
  Surv(TIME_DAYS, INFECTED) ~ ARM + AGE + SEX + COMORBIDITY,
  data = vaccine_trial
)

summary(cox_fit)

cox_results <- broom::tidy(cox_fit, exponentiate = TRUE, conf.int = TRUE) %>%
  mutate(
    VE_PERCENT = (1 - estimate) * 100,
    VE_CI_LOWER = (1 - conf.high) * 100,
    VE_CI_UPPER = (1 - conf.low) * 100
  ) %>%
  select(term, estimate, conf.low, conf.high, p.value, VE_PERCENT, VE_CI_LOWER, VE_CI_UPPER)

# Display results for ARM variable only
arm_result <- cox_results %>% filter(grepl("ARM", term))
cat("\n=== ADJUSTED VACCINE EFFICACY (Cox Model) ===\n")
cat(sprintf(
  "Hazard Ratio (VacrinA vs Placebo): %.3f (95%% CI: %.3f - %.3f)\n",
  arm_result$estimate, arm_result$conf.low, arm_result$conf.high
))
cat(sprintf(
  "Adjusted VE: %.1f%% (95%% CI: %.1f%% - %.1f%%)\n",
  arm_result$VE_PERCENT, arm_result$VE_CI_LOWER, arm_result$VE_CI_UPPER
))

# ==== Proportional Hazards Assumption Test ====

ar_vaccine <- ve_calc$ATTACK_RATE[1]
ar_placebo <- ve_calc$ATTACK_RATE[2]

ci_vaccine <- vaccine_binom$conf.int
ci_placebo <- placebo_binom$conf.int

# ==== Calculate Vaccine Efficacy ====
ve_point <- 1 - (ar_vaccine / ar_placebo)

# Exact CI for VE using delta method approximation
# (Conservative approach using outer bounds)
ve_ci_lower <- 1 - (ci_vaccine[2] / ci_placebo[1])
ve_ci_upper <- 1 - (ci_vaccine[1] / ci_placebo[2])

cat("\n=== VACCINE EFFICACY (Exact Method) ===\n")
cat(sprintf("VaccinA Attack Rate: %.2f%% (95%% CI: %.2f%% - %.2f%%)\n",
            ar_vaccine * 100, ci_vaccine[1] * 100, ci_vaccine[2] * 100))
cat(sprintf("Placebo Attack Rate: %.2f%% (95%% CI: %.2f%% - %.2f%%)\n",
            ar_placebo * 100, ci_placebo[1] * 100, ci_placebo[2] * 100))
cat(sprintf("\nVaccine Efficacy (VE): %.1f%% (95%% CI: %.1f%% - %.1f%%)\n",
            ve_point * 100, ve_ci_lower * 100, ve_ci_upper * 100))

# ==== VE Against Severe Disease ====
severe_data <- vaccine_trial %>%
  filter(INFECTED == 1) %>%
  group_by(ARM) %>%
  summarise(
    N_INFECTED = n(),
    N_SEVERE = sum(SEVERE),
    SEVERE_RATE = sum(SEVERE) / n(),
    .groups = "drop"
  )

print(severe_data)

severe_vaccine <- binom.test(
  severe_data$N_SEVERE[1],
  severe_data$N_INFECTED[1]
)
severe_placebo <- binom.test(
  severe_data$N_SEVERE[2],
  severe_data$N_INFECTED[2]
)

sr_vaccine <- severe_data$SEVERE_RATE[1]
sr_placebo <- severe_data$SEVERE_RATE[2]
ve_severe <- 1 - (sr_vaccine / sr_placebo)

cat("\n=== VACCINE EFFICACY AGAINST SEVERE DISEASE ===\n")
cat(sprintf("VE vs Severe Disease: %.1f%%\n", ve_severe * 100))
```

## Clinical Interpretation

A vaccine efficacy of ~70% against any infection means the vaccine reduces the absolute risk by roughly 70%. Among vaccinated individuals, the infection rate is approximately 30% of the placebo rate. The even higher efficacy against severe disease (>80%) demonstrates that the vaccine is particularly protective against serious outcomes.
:::

## 25.6 Antibody Titres: Longitudinal Immunogenicity Analysis

We analyze the immunogenicity response by comparing geometric mean antibody titres (GMT) over time and fitting a mixed-effects model.

```{r}
#| label: immunogenicity-analysis
#| eval: true

# ==== Calculate GMT and 95% CIs at Each Timepoint ====
gmt_summary <- ab_data %>%
  group_by(ARM, DAY) %>%
  summarise(
    N = n(),
    MEAN_LOG_TITRE = mean(LOG_TITRE, na.rm = TRUE),
    SD_LOG_TITRE = sd(LOG_TITRE, na.rm = TRUE),
    GMT = exp(MEAN_LOG_TITRE),
    SE = SD_LOG_TITRE / sqrt(N),
    GMT_CI_LOWER = exp(MEAN_LOG_TITRE - 1.96 * SE),
    GMT_CI_UPPER = exp(MEAN_LOG_TITRE + 1.96 * SE),
    .groups = "drop"
  )

print(gmt_summary)

# ==== Seroconversion: ≥4-fold Rise from Baseline ====
baseline_titres <- ab_data %>%
  filter(DAY == 0) %>%
  select(SUBJID, BASELINE_LOG_TITRE = LOG_TITRE)

seroconv_data <- ab_data %>%
  filter(DAY == 28) %>%  # Peak response timepoint
  left_join(baseline_titres, by = "SUBJID") %>%
  mutate(
    FOLD_RISE = exp(LOG_TITRE - BASELINE_LOG_TITRE),
    SEROCONVERTED = as.integer(FOLD_RISE >= 4)
  )

seroconv_summary <- seroconv_data %>%
  group_by(ARM) %>%
  summarise(
    N = n(),
    N_SEROCONVERTED = sum(SEROCONVERTED),
    SEROCONV_RATE = sum(SEROCONVERTED) / n(),
    SEROCONV_PERCENT = 100 * sum(SEROCONVERTED) / n(),
    .groups = "drop"
  )

print(seroconv_summary)

# Chi-square test for seroconversion rate difference
seroconv_table <- table(seroconv_data$ARM, seroconv_data$SEROCONVERTED)
seroconv_chi <- chisq.test(seroconv_table)
cat("\nSeroconversion Chi-square Test:\n")
print(seroconv_chi)
```

::: {.callout-note}
## GMT Interpretation

The VacrinA arm shows a robust and sustained antibody response, with GMT rising from ~7 at baseline to ~400 at Day 28, then waning to ~150 by Day 90. The placebo arm remains near baseline (~8) throughout, confirming lack of antigenic stimulation. A GMT ≥100 is typically associated with protective immunity.
:::

## 25.7 Correlates of Protection

We investigate whether antibody titre at peak response (Day 28) is predictive of protection against infection.

```{r}
#| label: correlates-protection
#| eval: true

# ==== Merge Day 28 Titre with Infection Outcome ====
day28_titres <- ab_data %>%
  filter(DAY == 28) %>%
  select(SUBJID, LOG_TITRE_D28 = LOG_TITRE, TITRE_D28 = TITRE)

outcome_data <- vaccine_trial %>%
  select(SUBJID, ARM, AGE, INFECTED) %>%
  left_join(day28_titres, by = "SUBJID")

# ==== Logistic Regression: Titre as Predictor ====
logistic_fit <- glm(
  INFECTED ~ LOG_TITRE_D28 + ARM + AGE,
  data = outcome_data,
  family = binomial(link = "logit")
)

summary(logistic_fit)

# ==== Extract Effect Sizes ====
logistic_coef <- broom::tidy(logistic_fit) %>%
  mutate(
    OR = exp(estimate),
    CI_LOWER = exp(estimate - 1.96 * std.error),
    CI_UPPER = exp(estimate + 1.96 * std.error)
  )

print(logistic_coef)

cat("\nInterpretation: Each 1-unit increase in log(titre) decreases odds of infection\n")
cat("by a factor of", round(logistic_coef$OR[2], 3), "\n")

# ==== Identify Protective Threshold ====
# Titre level associated with 70% efficacy (OR ~0.3)
threshold_log <- -log(0.3) / logistic_coef$estimate[2]
threshold_titre <- exp(threshold_log)

cat("\nEstimated Protective Titre Threshold (VE ≥ 70%):\n")
cat(sprintf("Log-titre: %.2f\n", threshold_log))
cat(sprintf("Titre: %.0f mIU/mL\n", threshold_titre))
```

This analysis reveals that higher antibody titres are strongly protective. Subjects with titre levels above the estimated threshold achieve the desired 70% vaccine efficacy target.
