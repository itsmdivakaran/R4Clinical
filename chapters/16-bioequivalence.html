27  Bioequivalence Studies

27.1 Introduction

Bioequivalence (BE) studies form the regulatory backbone for demonstrating that generic drugs are therapeutically equivalent to their reference (brand-name) counterparts. When a new drug developer wants to bring a generic version of an already-approved medicine to market, regulators (FDA, EMA, Health Canada) require pharmacokinetic (PK) evidence that the generic and reference formulations behave similarly in the human body. This is not a clinical efficacy trial—it is a targeted PK study measuring blood concentrations of the active pharmaceutical ingredient over time.

In this chapter, you will learn what bioequivalence means, why it matters, how to design and analyze BE studies, and how to implement the complete analysis in R. We will cover the regulatory framework, the mathematical foundations, and practical analysis with real data.

27.2 16.1 What Is Bioequivalence?

27.2.1 Clinical Context

Imagine a patient taking a blood pressure medication (lisinopril) manufactured by Merck. The drug is absorbed into the bloodstream, reaches peak concentration (\(C_{max}\)), and is gradually eliminated. The total drug exposure is measured by the area under the concentration-time curve (AUC). After the patent expires, another manufacturer (e.g., Teva) wants to produce a generic lisinopril tablet. The question is: will the generic tablet deliver the same drug exposure as the Merck tablet?

Bioequivalence means that two formulations (test and reference) deliver the same amount of active drug to the systemic circulation at the same rate. More formally:

  • Average Bioequivalence (ABE): The mean pharmacokinetic parameters of the test and reference formulations are statistically similar.
  • Population Bioequivalence: The variability of PK parameters is also comparable.
  • Individual Bioequivalence: Individuals can be switched between formulations without loss of efficacy.

For most regulatory purposes, the standard is Average Bioequivalence.

27.2.2 Key PK Parameters

The main metrics in a bioequivalence study are:

  1. \(C_{max}\): Peak (maximum) plasma concentration of drug
  2. \(t_{max}\): Time to reach \(C_{max}\)
  3. AUC₀₋ₜ: Area under curve from time 0 to last measured time
  4. AUC₀₋∞: Area under curve from 0 to infinity (extrapolated)
  5. \(t_{1/2}\): Half-life of the drug (elimination phase)

Among these, FDA and EMA guidances focus on \(C_{max}\) and AUC₀₋ₜ as the primary parameters.

27.2.3 The Regulatory Standard

Most regulatory agencies (FDA, EMA) define bioequivalence as:

\[\text{Bioequivalent if } 0.80 \leq \frac{GMR_{test}}{GMR_{ref}} \leq 1.25\]

where GMR is the geometric mean ratio. More precisely, we construct a 90% confidence interval (not 95%!) around the log-transformed ratio of geometric means. If this 90% CI falls entirely within [0.80, 1.25], bioequivalence is concluded.

Why 90% CI and not 95%? The FDA uses 90% CI to balance Type I and Type II errors; a 95% CI would be too conservative (harder to show BE) given that BE is a two-sided equivalence test.

Why log-transformation? PK parameters are typically log-normally distributed (right-skewed). Log-transformation stabilizes variance and makes them approximately normal, which is required for the statistical test.

27.3 16.2 The Crossover Study Design

27.3.1 Two-Sequence, Two-Period (2x2) Crossover

The gold standard for bioequivalence studies is the 2×2 crossover design:

  • Period 1: Subject receives either Test (T) or Reference (R) formulation
  • Washout period: Typically 1–4 weeks (depends on drug half-life) to eliminate carryover
  • Period 2: Subject receives the opposite formulation

Example randomization: - 6 subjects: Sequence 1 (R → T) - 6 subjects: Sequence 2 (T → R)

Blood samples are collected at multiple time points (e.g., 0, 0.5, 1, 2, 3, 4, 6, 8, 12, 24, 48 hours).

27.3.2 Statistical Model

The mixed-effects model for BE data is:

\[Y_{ijk} = \mu + S_{ik} + P_j + F_{j,k} + C_{j,k} + \epsilon_{ijk}\]

where: - \(Y_{ijk}\) = log-transformed PK parameter (AUC or Cmax) for subject \(i\), sequence \(k\), period \(j\) - \(\mu\) = overall intercept - \(S_{ik}\) = fixed effect of sequence \(k\) nested in subject \(i\) (subject is random) - \(P_j\) = fixed effect of period \(j\) (period effect) - \(F_{j,k}\) = fixed effect of formulation (test vs reference) - \(C_{j,k}\) = fixed effect of carryover (sequence-specific carry-over effect) - \(\epsilon_{ijk}\) = residual error

The fixed effects of interest are \(F_{1,R}\) and \(F_{2,T}\), which represent deviations from the reference.

27.4 16.3 Area Under the Curve (AUC) Calculation

27.4.1 The Trapezoidal Rule

AUC is rarely measured directly; instead, it is calculated from discrete plasma concentration measurements using numerical integration. The most common method is the trapezoidal rule:

\[AUC = \sum_{i=1}^{n-1} \frac{(C_i + C_{i+1})}{2} \cdot (t_{i+1} - t_i)\]

where \(C_i\) and \(C_{i+1}\) are consecutive concentrations and \((t_{i+1} - t_i)\) is the time interval.

Example: If concentration measurements are: - \(t=0\): \(C=0\) - \(t=1\): \(C=50\) - \(t=2\): \(C=75\) - \(t=4\): \(C=40\)

Then: \[AUC = \frac{(0+50)}{2}(1-0) + \frac{(50+75)}{2}(2-1) + \frac{(75+40)}{2}(4-2) = 25 + 62.5 + 115 = 202.5\]

27.4.2 Extrapolation to Infinity

For AUC₀₋∞, we must extrapolate beyond the last measured concentration. This is typically done by fitting an exponential to the terminal (elimination) phase and using:

\[AUC_{0-\infty} = AUC_{0-t} + \frac{C_t}{\lambda_z}\]

where \(\lambda_z\) is the terminal elimination rate constant (estimated from the slope of log(C) vs time in the terminal phase).

27.5 16.4 The TOST Procedure

27.5.1 Two One-Sided Tests (TOST)

Bioequivalence is conventionally tested using the Two One-Sided Tests (TOST) procedure:

\[H_{01}: \frac{\theta_{test}}{\theta_{ref}} \leq 0.80 \quad \text{vs} \quad H_{a1}: \frac{\theta_{test}}{\theta_{ref}} > 0.80\]

\[H_{02}: \frac{\theta_{test}}{\theta_{ref}} \geq 1.25 \quad \text{vs} \quad H_{a2}: \frac{\theta_{test}}{\theta_{ref}} < 1.25\]

We perform both tests at \(\alpha = 0.05\). Bioequivalence is concluded only if both tests reject their null hypotheses.

Equivalently: Construct the 90% confidence interval for the log-ratio of geometric means. If the entire CI falls within \([\log(0.80), \log(1.25)] = [-0.223, 0.223]\), we conclude bioequivalence.

27.5.2 Critical t-Value

For a crossover with \(N\) subjects and \(df = N - 2\) (approximately, after accounting for sequence and period effects), the critical t-value is \(t_{0.05, df}\).

27.6 16.5 Power Calculation for BE Studies

Sample size in BE studies is typically computed to achieve 80% power. The formula is:

\[n = \frac{2 \cdot (z_{\alpha/2} + z_\beta)^2 \cdot CV^2}{(\log 1.25)^2}\]

where: - \(z_{\alpha/2} = 1.96\) (two-sided \(\alpha = 0.05\)) - \(z_\beta = 0.84\) (80% power, so \(\beta = 0.20\)) - \(CV\) = coefficient of variation in log-scale (from prior studies or assumptions)

For example, if \(CV = 0.25\) (25% within-subject variability on log-scale):

\[n = \frac{2 \cdot (1.96 + 0.84)^2 \cdot (0.25)^2}{(0.223)^2} = \frac{2 \cdot 7.84 \cdot 0.0625}{0.0497} \approx 39 \text{ subjects}\]

27.7 16.6 Practical Analysis in R

27.7.1 Simulating a Bioequivalence Dataset

Let us create a realistic crossover BE study with 12 subjects, measuring AUC and Cmax for two formulations (Test and Reference).

set.seed(42)

# Simulate a 2x2 crossover bioequivalence study
# 12 subjects: 6 in each sequence

n_subjects <- 12
subjects <- rep(1:n_subjects, 2)  # each subject measured twice
period <- rep(c(1, 2), each = n_subjects)

# Define sequences (Sequence 1: R-T; Sequence 2: T-R)
sequence <- c(rep(1, 6), rep(2, 6))
sequence_rep <- rep(sequence, 2)

# Assign formulations based on sequence and period
formulation <- ifelse(
  (sequence_rep == 1 & period == 1) | (sequence_rep == 2 & period == 2),
  "Reference",
  "Test"
)

# True log-scale parameters
log_auc_ref <- log(500)   # Reference AUC mean in log scale
log_cmax_ref <- log(100)  # Reference Cmax mean in log scale

# Simulate AUC (subject effect + formulation effect + error)
subject_effect_auc <- rnorm(n_subjects, mean = 0, sd = 0.15)
formulation_effect <- ifelse(formulation == "Test", log(1.05), 0)  # 5% higher for Test
within_subject_var <- rnorm(n_subjects * 2, mean = 0, sd = 0.12)

log_auc <- log_auc_ref + rep(subject_effect_auc, 2) + formulation_effect + within_subject_var
auc <- exp(log_auc)

# Simulate Cmax similarly
subject_effect_cmax <- rnorm(n_subjects, mean = 0, sd = 0.18)
formulation_effect_cmax <- ifelse(formulation == "Test", log(1.08), 0)  # 8% higher for Test
within_subject_var_cmax <- rnorm(n_subjects * 2, mean = 0, sd = 0.15)

log_cmax <- log_cmax_ref + rep(subject_effect_cmax, 2) + formulation_effect_cmax + within_subject_var_cmax
cmax <- exp(log_cmax)

# Create dataset
be_data <- data.frame(
  subject = subjects,
  sequence = sequence_rep,
  period = period,
  formulation = formulation,
  auc = auc,
  cmax = cmax,
  log_auc = log_auc,
  log_cmax = log_cmax
)

} else {

}

27.7.6 Mean Plasma Concentration-Time Curves

For regulatory submissions, it is standard to show mean (or median) concentration-time curves for both formulations.

library(dplyr)

# Simulate concentration-time data for visualization
set.seed(42)

# Define sampling times (hours post-dose)
times <- c(0, 0.5, 1, 2, 3, 4, 6, 8, 12, 24)

# For each subject and formulation, generate a PK profile
# Using a simple one-compartment model: C(t) = (Dose/Vd) * exp(-k*t)
# Dose = 500 mg, Vd varies by subject, k varies by formulation (slightly)

pk_profiles <- data.frame()

for (subj in 1:n_subjects) {
  for (form in c("Reference", "Test")) {
    # Slightly different absorption and elimination for Test
    k_ref <- 0.25   # elimination rate (1/hour)
    ka_ref <- 1.2   # absorption rate

    if (form == "Test") {
      ka <- ka_ref * 0.95  # slightly slower absorption
      k <- k_ref * 1.05    # slightly faster elimination
    } else {
      ka <- ka_ref
      k <- k_ref
    }

    # Simple model: delayed absorption then exponential decay
    Vd <- 50 + rnorm(1, 0, 5)
    dose <- 500
    Cmax_theo <- (dose / Vd) * (ka / (ka - k))

    for (t in times) {
      if (t == 0) {
        conc <- 0
      } else {
        conc <- (dose / Vd) * (ka / (ka - k)) * (exp(-k * t) - exp(-ka * t))
      }
      conc <- conc + rnorm(1, 0, conc * 0.05)  # 5% measurement error
      conc <- max(conc, 0)

      pk_profiles <- rbind(pk_profiles, data.frame(
        subject = subj,
        formulation = form,
        time = t,
        concentration = conc
      ))
    }
  }
}

# Compute mean concentration at each time point for each formulation
mean_pk <- pk_profiles %>%
  group_by(time, formulation) %>%
  summarise(mean_conc = mean(concentration), .groups = "drop")

# Plot

# Bioequivalence Studies


## Introduction

Bioequivalence (BE) studies form the regulatory backbone for demonstrating that generic drugs are therapeutically equivalent to their reference (brand-name) counterparts. When a new drug developer wants to bring a generic version of an already-approved medicine to market, regulators (FDA, EMA, Health Canada) require pharmacokinetic (PK) evidence that the generic and reference formulations behave similarly in the human body. This is not a clinical efficacy trial—it is a targeted PK study measuring blood concentrations of the active pharmaceutical ingredient over time.

In this chapter, you will learn what bioequivalence means, why it matters, how to design and analyze BE studies, and how to implement the complete analysis in R. We will cover the regulatory framework, the mathematical foundations, and practical analysis with real data.

## 16.1 What Is Bioequivalence?

### Clinical Context

Imagine a patient taking a blood pressure medication (lisinopril) manufactured by Merck. The drug is absorbed into the bloodstream, reaches peak concentration ($C_{max}$), and is gradually eliminated. The total drug exposure is measured by the area under the concentration-time curve (AUC). After the patent expires, another manufacturer (e.g., Teva) wants to produce a generic lisinopril tablet. The question is: will the generic tablet deliver the same drug exposure as the Merck tablet?

**Bioequivalence** means that two formulations (test and reference) deliver the same amount of active drug to the systemic circulation at the same rate. More formally:

- **Average Bioequivalence (ABE):** The mean pharmacokinetic parameters of the test and reference formulations are statistically similar.
- **Population Bioequivalence:** The variability of PK parameters is also comparable.
- **Individual Bioequivalence:** Individuals can be switched between formulations without loss of efficacy.

For most regulatory purposes, the standard is **Average Bioequivalence**.

### Key PK Parameters

The main metrics in a bioequivalence study are:

1. **$C_{max}$:** Peak (maximum) plasma concentration of drug
2. **$t_{max}$:** Time to reach $C_{max}$
3. **AUC₀₋ₜ:** Area under curve from time 0 to last measured time
4. **AUC₀₋∞:** Area under curve from 0 to infinity (extrapolated)
5. **$t_{1/2}$:** Half-life of the drug (elimination phase)

Among these, FDA and EMA guidances focus on **$C_{max}$** and **AUC₀₋ₜ** as the primary parameters.

### The Regulatory Standard

Most regulatory agencies (FDA, EMA) define bioequivalence as:

$$\text{Bioequivalent if } 0.80 \leq \frac{GMR_{test}}{GMR_{ref}} \leq 1.25$$

where GMR is the **geometric mean ratio**. More precisely, we construct a **90% confidence interval** (not 95%!) around the log-transformed ratio of geometric means. If this 90% CI falls entirely within [0.80, 1.25], bioequivalence is concluded.

**Why 90% CI and not 95%?** The FDA uses 90% CI to balance Type I and Type II errors; a 95% CI would be too conservative (harder to show BE) given that BE is a two-sided equivalence test.

**Why log-transformation?** PK parameters are typically log-normally distributed (right-skewed). Log-transformation stabilizes variance and makes them approximately normal, which is required for the statistical test.

## 16.2 The Crossover Study Design

### Two-Sequence, Two-Period (2x2) Crossover

The gold standard for bioequivalence studies is the **2×2 crossover design**:

- **Period 1:** Subject receives either Test (T) or Reference (R) formulation
- **Washout period:** Typically 1–4 weeks (depends on drug half-life) to eliminate carryover
- **Period 2:** Subject receives the opposite formulation

Example randomization:
- 6 subjects: Sequence 1 (R → T)
- 6 subjects: Sequence 2 (T → R)

Blood samples are collected at multiple time points (e.g., 0, 0.5, 1, 2, 3, 4, 6, 8, 12, 24, 48 hours).

### Statistical Model

The mixed-effects model for BE data is:

$$Y_{ijk} = \mu + S_{ik} + P_j + F_{j,k} + C_{j,k} + \epsilon_{ijk}$$

where:
- $Y_{ijk}$ = log-transformed PK parameter (AUC or Cmax) for subject $i$, sequence $k$, period $j$
- $\mu$ = overall intercept
- $S_{ik}$ = fixed effect of sequence $k$ nested in subject $i$ (subject is random)
- $P_j$ = fixed effect of period $j$ (period effect)
- $F_{j,k}$ = fixed effect of formulation (test vs reference)
- $C_{j,k}$ = fixed effect of carryover (sequence-specific carry-over effect)
- $\epsilon_{ijk}$ = residual error

The fixed effects of interest are **$F_{1,R}$ and $F_{2,T}$**, which represent deviations from the reference.

## 16.3 Area Under the Curve (AUC) Calculation

### The Trapezoidal Rule

AUC is rarely measured directly; instead, it is calculated from discrete plasma concentration measurements using numerical integration. The most common method is the **trapezoidal rule**:

$$AUC = \sum_{i=1}^{n-1} \frac{(C_i + C_{i+1})}{2} \cdot (t_{i+1} - t_i)$$

where $C_i$ and $C_{i+1}$ are consecutive concentrations and $(t_{i+1} - t_i)$ is the time interval.

**Example:** If concentration measurements are:
- $t=0$: $C=0$
- $t=1$: $C=50$
- $t=2$: $C=75$
- $t=4$: $C=40$

Then:
$$AUC = \frac{(0+50)}{2}(1-0) + \frac{(50+75)}{2}(2-1) + \frac{(75+40)}{2}(4-2) = 25 + 62.5 + 115 = 202.5$$

### Extrapolation to Infinity

For AUC₀₋∞, we must extrapolate beyond the last measured concentration. This is typically done by fitting an exponential to the terminal (elimination) phase and using:

$$AUC_{0-\infty} = AUC_{0-t} + \frac{C_t}{\lambda_z}$$

where $\lambda_z$ is the terminal elimination rate constant (estimated from the slope of log(C) vs time in the terminal phase).

## 16.4 The TOST Procedure

### Two One-Sided Tests (TOST)

Bioequivalence is conventionally tested using the **Two One-Sided Tests (TOST)** procedure:

$$H_{01}: \frac{\theta_{test}}{\theta_{ref}} \leq 0.80 \quad \text{vs} \quad H_{a1}: \frac{\theta_{test}}{\theta_{ref}} > 0.80$$

$$H_{02}: \frac{\theta_{test}}{\theta_{ref}} \geq 1.25 \quad \text{vs} \quad H_{a2}: \frac{\theta_{test}}{\theta_{ref}} < 1.25$$

We perform both tests at $\alpha = 0.05$. Bioequivalence is concluded **only if both tests reject their null hypotheses**.

**Equivalently:** Construct the 90% confidence interval for the log-ratio of geometric means. If the entire CI falls within $[\log(0.80), \log(1.25)] = [-0.223, 0.223]$, we conclude bioequivalence.

### Critical t-Value

For a crossover with $N$ subjects and $df = N - 2$ (approximately, after accounting for sequence and period effects), the critical t-value is $t_{0.05, df}$.

## 16.5 Power Calculation for BE Studies

Sample size in BE studies is typically computed to achieve 80% power. The formula is:

$$n = \frac{2 \cdot (z_{\alpha/2} + z_\beta)^2 \cdot CV^2}{(\log 1.25)^2}$$

where:
- $z_{\alpha/2} = 1.96$ (two-sided $\alpha = 0.05$)
- $z_\beta = 0.84$ (80% power, so $\beta = 0.20$)
- $CV$ = coefficient of variation in log-scale (from prior studies or assumptions)

For example, if $CV = 0.25$ (25% within-subject variability on log-scale):

$$n = \frac{2 \cdot (1.96 + 0.84)^2 \cdot (0.25)^2}{(0.223)^2} = \frac{2 \cdot 7.84 \cdot 0.0625}{0.0497} \approx 39 \text{ subjects}$$

### Simulating a Bioequivalence Dataset

```{r}
set.seed(42)

# Simulate a 2x2 crossover bioequivalence study
# 12 subjects: 6 in each sequence

n_subjects <- 12
subjects <- rep(1:n_subjects, 2)  # each subject measured twice
period <- rep(c(1, 2), each = n_subjects)

# Define sequences (Sequence 1: R-T; Sequence 2: T-R)
sequence <- c(rep(1, 6), rep(2, 6))
sequence_rep <- rep(sequence, 2)

# Assign formulations based on sequence and period
formulation <- ifelse(
  (sequence_rep == 1 & period == 1) | (sequence_rep == 2 & period == 2),
  "Reference",
  "Test"
)

# True log-scale parameters
log_auc_ref <- log(500)   # Reference AUC mean in log scale
log_cmax_ref <- log(100)  # Reference Cmax mean in log scale

# Simulate AUC (subject effect + formulation effect + error)
subject_effect_auc <- rnorm(n_subjects, mean = 0, sd = 0.15)
formulation_effect <- ifelse(formulation == "Test", log(1.05), 0)  # 5% higher for Test
within_subject_var <- rnorm(n_subjects * 2, mean = 0, sd = 0.12)

log_auc <- log_auc_ref + rep(subject_effect_auc, 2) + formulation_effect + within_subject_var
auc <- exp(log_auc)

# Simulate Cmax similarly
subject_effect_cmax <- rnorm(n_subjects, mean = 0, sd = 0.18)
formulation_effect_cmax <- ifelse(formulation == "Test", log(1.08), 0)  # 8% higher for Test
within_subject_var_cmax <- rnorm(n_subjects * 2, mean = 0, sd = 0.15)

log_cmax <- log_cmax_ref + rep(subject_effect_cmax, 2) + formulation_effect_cmax + within_subject_var_cmax
cmax <- exp(log_cmax)

# Create dataset

  subject = subjects,
  sequence = sequence_rep,
```

### Linear Model Analysis

```{r}
# For AUC: fit mixed model using lm with sequence as fixed effect
# (In practice, use lme4::lmer for random subject effect)

# Fit linear model with fixed effects: sequence, period, formulation
# Log-transform the response

model_auc <- lm(log_auc ~ factor(sequence) + factor(period) + formulation + factor(subject),
                data = be_data)

summary(model_auc)

# Extract the log-scale difference between Test and Reference
# The coefficient for "formulationTest" represents log(Test) - log(Reference)
log_diff_auc <- coef(model_auc)["formulationTest"]
se_log_diff_auc <- sqrt(diag(vcov(model_auc)))["formulationTest"]

cat("\n=== AUC Bioequivalence Analysis ===\n")
cat("Log-scale difference (Test - Ref):", round(log_diff_auc, 4), "\n")
cat("Standard Error:", round(se_log_diff_auc, 4), "\n")

# Compute 90% CI for the log-ratio
# Using t-distribution with residual df
df_residual <- df.residual(model_auc)
t_crit <- qt(0.95, df_residual)  # one-sided 90% = two-sided 80%?
# Actually, for 90% CI: use qt(0.95, df) for upper bound, qt(0.05, df) for lower
t_crit_90 <- qt(0.95, df_residual)

ci_lower_log <- log_diff_auc - t_crit_90 * se_log_diff_auc
ci_upper_log <- log_diff_auc + t_crit_90 * se_log_diff_auc

# Convert back to ratio scale
ci_lower_ratio <- exp(ci_lower_log)
ci_upper_ratio <- exp(ci_upper_log)

cat("\n90% CI for GMR (log-scale):",
    "[", round(ci_lower_log, 4), ", ", round(ci_upper_log, 4), "]\n")
cat("90% CI for GMR (ratio-scale):",
    "[", round(ci_lower_ratio, 4), ", ", round(ci_upper_ratio, 4), "]\n")

# Check BE criteria
be_lower <- 0.80
be_upper <- 1.25

be_conclusion_auc <- (ci_lower_ratio >= be_lower) && (ci_upper_ratio <= be_upper)

cat("\nBE Criteria: [0.80, 1.25]\n")
cat("CI within BE range?", be_conclusion_auc, "\n")
if (isTRUE(be_conclusion_auc)) {
  cat("CONCLUSION: Bioequivalence ESTABLISHED for AUC\n")
} else {
  cat("CONCLUSION: Bioequivalence NOT established for AUC\n")
}
```

```{r}
library(dplyr)

# Simulate concentration-time data for visualization
set.seed(42)

# Define sampling times (hours post-dose)
times <- c(0, 0.5, 1, 2, 3, 4, 6, 8, 12, 24)

# For each subject and formulation, generate a PK profile
# Using a simple one-compartment model: C(t) = (Dose/Vd) * exp(-k*t)
# Dose = 500 mg, Vd varies by subject, k varies by formulation (slightly)

pk_profiles <- data.frame()

for (subj in 1:n_subjects) {
  for (form in c("Reference", "Test")) {
    # Slightly different absorption and elimination for Test
    k_ref <- 0.25   # elimination rate (1/hour)
    ka_ref <- 1.2   # absorption rate

    if (form == "Test") {
      ka <- ka_ref * 0.95  # slightly slower absorption
      k <- k_ref * 1.05    # slightly faster elimination
    } else {
      ka <- ka_ref
      k <- k_ref
    }

    # Simple model: delayed absorption then exponential decay
    Vd <- 50 + rnorm(1, 0, 5)
    dose <- 500
    Cmax_theo <- (dose / Vd) * (ka / (ka - k))

    for (t in times) {
      if (t == 0) {
        conc <- 0
      } else {
        conc <- (dose / Vd) * (ka / (ka - k)) * (exp(-k * t) - exp(-ka * t))
      }
      conc <- conc + rnorm(1, 0, conc * 0.05)  # 5% measurement error
      conc <- max(conc, 0)

      pk_profiles <- rbind(pk_profiles, data.frame(
        subject = subj,
        formulation = form,
        time = t,
        concentration = conc
      ))
    }
  }
}

# Compute mean concentration at each time point for each formulation
mean_pk <- pk_profiles %>%
  group_by(time, formulation) %>%
  summarise(mean_conc = mean(concentration), .groups = "drop")
```
