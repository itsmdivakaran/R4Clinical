30  Real-World Evidence (RWE)

30.1 Introduction

Randomized controlled trials are the gold standard for assessing drug efficacy. Yet most clinical decisions in everyday practice are made on observational data: electronic health records, patient registries, administrative claims, and patient-reported outcomes. The question is: how can we learn causal effects from observational data when patients are not randomly assigned to treatments?

Real-world evidence (RWE) uses observational data to understand treatment effectiveness, safety, and value in routine clinical practice. This chapter covers the challenges of observational data (confounding, selection bias), propensity score methods for causal inference, instrumental variables, difference-in-differences designs, and regulatory guidance on RWE.

30.2 19.1 What Is Real-World Evidence?

30.2.1 Sources of RWE

Electronic Health Records (EHR): Detailed clinical data including diagnoses, lab values, medications, and outcomes. Covers entire patient journeys but may have missing data and measurement error.

Administrative/Claims Data: Insurance claims, hospital discharge records. Large sample sizes but limited clinical detail.

Patient Registries: Disease-specific (cancer registry, diabetes registry) or procedure-specific (stent registry). Rich clinical detail but potential selection bias.

Patient-Reported Outcomes (PRO): Symptoms, quality of life, adherence. Direct patient voice but subject to recall bias.

Wearables and Digital Health: Continuous monitoring of activity, heart rate, sleep. Novel but may have missing data.

30.2.2 RWE vs. RCT: Trade-Offs

Aspect RCT RWE
Eligibility Strict inclusion/exclusion criteria Reflects real population (heterogeneous)
Treatment Assignment Random (balanced covariates) Based on physician/patient choice (confounded)
Measurement Standardized, frequent Clinical routine, variable
Sample Size Often limited (1,000–5,000) Often large (10,000–1,000,000+)
Follow-up Structured, high adherence Variable, reflects real-world
Cost High Often lower (secondary data)
Generalizability Limited (selected population) Higher (typical patients)
Causal Inference Strong (randomization) Weak (must adjust for confounding)

30.3 19.2 The Causal Framework: Confounding and the Counterfactual

30.3.1 Confounding

A variable \(C\) is a confounder if it: 1. Affects the treatment assignment (patients with condition X are more likely to receive drug A) 2. Affects the outcome independent of treatment 3. Is not in the causal path between treatment and outcome

Example: In observational data on cholesterol medications, age is a confounder. Older patients are more likely to be prescribed statins AND have higher risk of heart disease. If we ignore age, we may incorrectly attribute the heart disease risk to statins.

30.3.2 The Counterfactual Framework

Causal effect is defined as the difference between two counterfactual outcomes:

\[\text{Causal Effect for person } i = Y_i(1) - Y_i(0)\]

where: - \(Y_i(1)\) = outcome if person \(i\) receives treatment - \(Y_i(0)\) = outcome if person \(i\) receives control

The fundamental problem of causal inference is that we never observe both \(Y_i(1)\) and \(Y_i(0)\) for the same person; we observe only one.

30.3.3 Treatment Effect Parameters

  • Average Treatment Effect (ATE): \(E[Y(1) - Y(0)]\) (effect in entire population)
  • Average Treatment Effect on the Treated (ATT): \(E[Y(1) - Y(0) | T=1]\) (effect in treated patients)
  • Conditional Average Treatment Effect (CATE): \(E[Y(1) - Y(0) | X]\) (effect heterogeneous by covariates \(X\))

30.4 19.3 Propensity Score Methods

30.4.1 The Propensity Score

The propensity score is the probability of receiving treatment given observed covariates:

\[e(X) = P(T = 1 | X) = \text{logit}^{-1}(\beta_0 + \beta_1 X_1 + \cdots + \beta_p X_p)\]

where \(T \in \{0, 1\}\) is the treatment indicator and \(X\) are observed covariates.

Fit using logistic regression.

30.4.2 Key Assumption: Strong Ignorability (Unconfoundedness)

Propensity score methods assume: \[Y(0), Y(1) \perp T | X\]

That is, conditional on observed covariates \(X\), treatment is as-if random. This assumes no unmeasured confounding—a strong assumption that cannot be tested from data alone.

30.4.3 Propensity Score Matching

Goal: Create matched samples where treated and control groups have similar covariate distributions.

Procedure: 1. Fit logistic regression to estimate propensity scores for all subjects 2. For each treated subject, find control subjects with similar propensity score (within a caliper, e.g., ±0.1) 3. Match 1:1 (one control per treated) or 1:k 4. Analyze the matched sample, comparing outcomes between groups

30.4.4 Inverse Probability of Treatment Weighting (IPTW)

Instead of matching, weight each observation:

\[w_i = \frac{T_i}{e(X_i)} + \frac{1 - T_i}{1 - e(X_i)}\]

  • Treated subjects with low propensity score (unlikely to be treated) get high weight
  • Control subjects with high propensity score (likely to be treated) get high weight
  • This creates a pseudo-population where treatment is independent of covariates

Fit a weighted regression (e.g., logistic regression with IPTW weights) to estimate treatment effect.

30.4.5 Overlap (Positivity) Condition

For both matching and IPTW to work, we need overlap: \(0 < e(X) < 1\) for all subjects. If some treated subjects have \(e(X) \approx 1\) (certain to be treated) or control subjects have \(e(X) \approx 0\), there is no comparable population, and causal inference is infeasible.

30.4.6 Assessing Balance

After matching or IPTW, check that treated and control groups have similar covariate distributions using Standardized Mean Difference (SMD):

\[\text{SMD} = \frac{|\bar{X}_{\text{treated}} - \bar{X}_{\text{control}}|}{\sqrt{(s_{\text{treated}}^2 + s_{\text{control}}^2) / 2}}\]

  • SMD < 0.10 indicates good balance
  • SMD > 0.20 indicates imbalance

A Love plot visualizes SMD for all covariates before and after matching/weighting.

30.5 19.4 Alternative Designs: Instrumental Variables and Difference-in-Differences

30.5.1 Instrumental Variables (Brief Overview)

An instrument \(Z\) is a variable that: - Affects treatment assignment (\(Z \to T\)) - Affects outcome only through treatment (\(Z \to Y\) only via \(T\)) - Is independent of unmeasured confounders

Example: Physician preference for a drug (due to, say, different training) as an instrument for patient treatment.

Instrumental variable estimation (2-stage least squares) can estimate treatment effect even with unmeasured confounding, under strong assumptions.

30.5.2 Difference-in-Differences (Brief Overview)

When you have data from two time periods and two groups (one receiving treatment, one not), difference-in-differences compares:

\[\text{Effect} = (Y_{\text{treated, after}} - Y_{\text{treated, before}}) - (Y_{\text{control, after}} - Y_{\text{control, before}})\]

This removes time-constant confounding. Useful for policy evaluations.

30.6 19.5 FDA Guidance on Real-World Evidence

The FDA’s Real-World Evidence Program (updated 2022) provides a framework for using RWE to: - Support approval of new indications - Address post-market safety questions - Inform labeling changes

Key FDA recommendations: 1. Study Design: Prospectively designed observational studies are preferred; retrospective analysis of EHR is lower quality 2. Data Quality: Document data sources, validation, and completeness 3. Confounding: Use propensity score or other adjustment methods 4. Transparency: Pre-specify analysis plan (registered in clinical trial database if possible) 5. Reproducibility: Share code and datasets when feasible 6. Comparison to RCT: When possible, validate RWE findings against RCT results

30.7 19.6 Practical Analysis in R

30.7.1 Example: Propensity Score Matching for Heart Failure Treatment

We simulate observational data: heart failure patients treated with ACE inhibitor (treatment=1) or beta-blocker (treatment=0), measured by 1-year mortality.

library(tidyverse)
library(MatchIt)  # for propensity score matching
library(cobalt)   # for balance assessment and Love plots

set.seed(42)

# Simulate heart failure cohort
n <- 500

# Confounders (unmeasured by RCT but observed in RWE)
age <- rnorm(n, mean = 65, sd = 12)
ejection_fraction <- rnorm(n, mean = 35, sd = 15)  # % LV ejection fraction
nyha_class <- sample(1:4, n, replace = TRUE, prob = c(0.3, 0.35, 0.25, 0.1))

# Propensity for ACE inhibitor (older, lower EF, higher NYHA get ACE inhibitor more)
ps_linear <- -3 + 0.05 * age - 0.08 * ejection_fraction + 0.3 * nyha_class
ps <- plogis(ps_linear)  # inverse logit

treatment <- rbinom(n, size = 1, prob = ps)

# Outcome: 1-year mortality
# Confounding: age and lower EF increase mortality
# Treatment effect: ACE inhibitor has protective effect
mortality_prob <- plogis(-4 + 0.05 * age - 0.02 * ejection_fraction - 0.8 * treatment)
mortality <- rbinom(n, size = 1, prob = mortality_prob)

# Create dataset
rwe_data <- data.frame(
  subject_id = 1:n,
  age = age,
  ejection_fraction = ejection_fraction,
  nyha_class = nyha_class,
  treatment = treatment,
  mortality = mortality
)

30.7.8 Checking Balance After IPTW

# Compute weighted means and SDs for balance assessment
balance_iptw <- rwe_data %>%
  pivot_longer(
    cols = c(age, ejection_fraction, nyha_class),
    names_to = "variable",
    values_to = "value"
  ) %>%
  group_by(variable, treatment) %>%
  summarise(
    weighted_mean = weighted.mean(value, w = iptw_weight),
    weighted_sd = sqrt(Hmisc::wtd.var(value, weights = iptw_weight)),
    .groups = "drop"
  ) %>%
  pivot_wider(
    names_from = treatment,
    values_from = c(weighted_mean, weighted_sd)
  ) %>%
  mutate(
    smd = (weighted_mean_1 - weighted_mean_0) /
          sqrt((weighted_sd_0^2 + weighted_sd_1^2) / 2)
  )

# - be moderately associated with treatment assignment

# Real-World Evidence (RWE)


## Introduction

Randomized controlled trials are the gold standard for assessing drug efficacy. Yet most clinical decisions in everyday practice are made on observational data: electronic health records, patient registries, administrative claims, and patient-reported outcomes. The question is: how can we learn causal effects from observational data when patients are not randomly assigned to treatments?

**Real-world evidence (RWE)** uses observational data to understand treatment effectiveness, safety, and value in routine clinical practice. This chapter covers the challenges of observational data (confounding, selection bias), propensity score methods for causal inference, instrumental variables, difference-in-differences designs, and regulatory guidance on RWE.

## 19.1 What Is Real-World Evidence?

### Sources of RWE

**Electronic Health Records (EHR):** Detailed clinical data including diagnoses, lab values, medications, and outcomes. Covers entire patient journeys but may have missing data and measurement error.

**Administrative/Claims Data:** Insurance claims, hospital discharge records. Large sample sizes but limited clinical detail.

**Patient Registries:** Disease-specific (cancer registry, diabetes registry) or procedure-specific (stent registry). Rich clinical detail but potential selection bias.

**Patient-Reported Outcomes (PRO):** Symptoms, quality of life, adherence. Direct patient voice but subject to recall bias.

**Wearables and Digital Health:** Continuous monitoring of activity, heart rate, sleep. Novel but may have missing data.

### RWE vs. RCT: Trade-Offs

| Aspect | RCT | RWE |
|--------|-----|-----|
| **Eligibility** | Strict inclusion/exclusion criteria | Reflects real population (heterogeneous) |
| **Treatment Assignment** | Random (balanced covariates) | Based on physician/patient choice (confounded) |
| **Measurement** | Standardized, frequent | Clinical routine, variable |
| **Sample Size** | Often limited (1,000–5,000) | Often large (10,000–1,000,000+) |
| **Follow-up** | Structured, high adherence | Variable, reflects real-world | |
| **Cost** | High | Often lower (secondary data) |
| **Generalizability** | Limited (selected population) | Higher (typical patients) |
| **Causal Inference** | Strong (randomization) | Weak (must adjust for confounding) |

## 19.2 The Causal Framework: Confounding and the Counterfactual

### Confounding

A variable $C$ is a **confounder** if it:
1. Affects the treatment assignment (patients with condition X are more likely to receive drug A)
2. Affects the outcome independent of treatment
3. Is not in the causal path between treatment and outcome

Example: In observational data on cholesterol medications, age is a confounder. Older patients are more likely to be prescribed statins AND have higher risk of heart disease. If we ignore age, we may incorrectly attribute the heart disease risk to statins.

### The Counterfactual Framework

**Causal effect** is defined as the difference between two **counterfactual** outcomes:

$$\text{Causal Effect for person } i = Y_i(1) - Y_i(0)$$

where:
- $Y_i(1)$ = outcome if person $i$ receives treatment
- $Y_i(0)$ = outcome if person $i$ receives control

The **fundamental problem of causal inference** is that we never observe both $Y_i(1)$ and $Y_i(0)$ for the same person; we observe only one.

### Treatment Effect Parameters

- **Average Treatment Effect (ATE):** $E[Y(1) - Y(0)]$ (effect in entire population)
- **Average Treatment Effect on the Treated (ATT):** $E[Y(1) - Y(0) | T=1]$ (effect in treated patients)
- **Conditional Average Treatment Effect (CATE):** $E[Y(1) - Y(0) | X]$ (effect heterogeneous by covariates $X$)

## 19.3 Propensity Score Methods

### The Propensity Score

The **propensity score** is the probability of receiving treatment given observed covariates:

$$e(X) = P(T = 1 | X) = \text{logit}^{-1}(\beta_0 + \beta_1 X_1 + \cdots + \beta_p X_p)$$

where $T \in \{0, 1\}$ is the treatment indicator and $X$ are observed covariates.

Fit using logistic regression.

### Key Assumption: Strong Ignorability (Unconfoundedness)

Propensity score methods assume:
$$Y(0), Y(1) \perp T | X$$

That is, conditional on observed covariates $X$, treatment is as-if random. This assumes **no unmeasured confounding**—a strong assumption that cannot be tested from data alone.

### Propensity Score Matching

**Goal:** Create matched samples where treated and control groups have similar covariate distributions.

**Procedure:**
1. Fit logistic regression to estimate propensity scores for all subjects
2. For each treated subject, find control subjects with similar propensity score (within a **caliper**, e.g., ±0.1)
3. Match 1:1 (one control per treated) or 1:k
4. Analyze the matched sample, comparing outcomes between groups

### Inverse Probability of Treatment Weighting (IPTW)

Instead of matching, **weight** each observation:

$$w_i = \frac{T_i}{e(X_i)} + \frac{1 - T_i}{1 - e(X_i)}$$

- Treated subjects with low propensity score (unlikely to be treated) get high weight
- Control subjects with high propensity score (likely to be treated) get high weight
- This creates a **pseudo-population** where treatment is independent of covariates

Fit a weighted regression (e.g., logistic regression with IPTW weights) to estimate treatment effect.

### Overlap (Positivity) Condition

For both matching and IPTW to work, we need overlap: $0 < e(X) < 1$ for all subjects. If some treated subjects have $e(X) \approx 1$ (certain to be treated) or control subjects have $e(X) \approx 0$, there is no comparable population, and causal inference is infeasible.

### Assessing Balance

After matching or IPTW, check that treated and control groups have similar covariate distributions using **Standardized Mean Difference (SMD)**:

$$\text{SMD} = \frac{|\bar{X}_{\text{treated}} - \bar{X}_{\text{control}}|}{\sqrt{(s_{\text{treated}}^2 + s_{\text{control}}^2) / 2}}$$

- SMD < 0.10 indicates good balance
- SMD > 0.20 indicates imbalance

A **Love plot** visualizes SMD for all covariates before and after matching/weighting.

## 19.4 Alternative Designs: Instrumental Variables and Difference-in-Differences

### Instrumental Variables (Brief Overview)

An **instrument** $Z$ is a variable that:
- Affects treatment assignment ($Z \to T$)
- Affects outcome only through treatment ($Z \to Y$ only via $T$)
- Is independent of unmeasured confounders

Example: Physician preference for a drug (due to, say, different training) as an instrument for patient treatment.

Instrumental variable estimation (2-stage least squares) can estimate treatment effect even with unmeasured confounding, under strong assumptions.

### Difference-in-Differences (Brief Overview)

When you have data from two time periods and two groups (one receiving treatment, one not), **difference-in-differences** compares:

$$\text{Effect} = (Y_{\text{treated, after}} - Y_{\text{treated, before}}) - (Y_{\text{control, after}} - Y_{\text{control, before}})$$

This removes time-constant confounding. Useful for policy evaluations.

## 19.5 FDA Guidance on Real-World Evidence

The FDA's **Real-World Evidence Program** (updated 2022) provides a framework for using RWE to:
- Support approval of new indications
- Address post-market safety questions
- Inform labeling changes

Key FDA recommendations:
1. **Study Design:** Prospectively designed observational studies are preferred; retrospective analysis of EHR is lower quality
2. **Data Quality:** Document data sources, validation, and completeness
3. **Confounding:** Use propensity score or other adjustment methods
4. **Transparency:** Pre-specify analysis plan (registered in clinical trial database if possible)
5. **Reproducibility:** Share code and datasets when feasible
6. **Comparison to RCT:** When possible, validate RWE findings against RCT results

## 19.8 Sensitivity Analysis for Unmeasured Confounding

Propensity score methods assume all confounders are measured. But unmeasured confounding could bias results. Conduct sensitivity analysis:

```{r}
# Simple sensitivity analysis: how strong would an unmeasured confounder need to be
# to flip our conclusion?

# Observed treatment effect: OR ≈ 0.45 (protective)
# For the effect to be null (OR = 1.0), an unmeasured confounder would need to:
# - be moderately associated with treatment assignment
# - be moderately associated with outcome

# Compute "E-value" (sensitivity parameter)
# E-value = OR + sqrt(OR * (OR - 1))
# Interpretation: unmeasured confounder would need this strength of association

observed_or <- iptw_or
e_value <- observed_or + sqrt(observed_or * (observed_or - 1))

cat("=== SENSITIVITY ANALYSIS: Unmeasured Confounding ===\n\n")
cat("Observed Effect (IPTW):", round(observed_or, 3), "\n")
cat("E-value:", round(e_value, 3), "\n\n")
cat("Interpretation:\n")
cat("To explain away the observed protective effect of the treatment,\n")
cat("an unmeasured confounder would need to have:\n")
cat("- At least", round(e_value, 2), "x association with treatment assignment, AND\n")
cat("- At least", round(e_value, 2), "x association with mortality\n")
cat("(Or some combination, e.g., 2.0x and 1.5x, whose product is e-value)\n\n")

# Check clinical plausibility
if (isTRUE(e_value > 3.0)) {
  cat("REASSURING: An unmeasured confounder would need implausibly strong associations\n")
  cat("to explain away the treatment effect.\n")
} else if (isTRUE(e_value > 1.5)) {
  cat("MODERATE: An unmeasured confounder with moderate-to-strong associations\n")
  cat("could plausibly explain the treatment effect.\n")
} else {
  cat("CONCERNING: A weak unmeasured confounder could explain the treatment effect.\n")
}
```
