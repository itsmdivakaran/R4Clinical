13  Non-Parametric Tests in Clinical Research

13.1 Introduction

Non-parametric (distribution-free) tests make fewer assumptions about the underlying data distribution. They are particularly valuable in clinical research when:

  • Sample sizes are small (n < 30) and normality cannot be verified
  • Data are ordinal (e.g., pain scores, quality-of-life ratings)
  • Distributions are heavily skewed or contain outliers
  • Outcomes are censored counts or time-to-event with small samples

This chapter covers:

  • Wilcoxon signed-rank test: Paired or one-sample comparisons
  • Mann–Whitney U test: Two independent groups
  • Kruskal–Wallis test: Multiple independent groups
  • Friedman test: Repeated measures across groups
  • Spearman correlation: Monotonic association between variables
  • Permutation tests: Exact non-parametric inference
  • Choosing between parametric and non-parametric approaches

13.2 1. When to Use Non-Parametric Tests

13.3 2. Wilcoxon Signed-Rank Test

13.3.1 2.1 Concept and Hypotheses

The Wilcoxon signed-rank test evaluates whether the median of paired differences (or a one-sample median) equals a hypothesised value. It replaces the paired t-test when the normality assumption fails.

Procedure: 1. Compute pairwise differences \(d_i = x_{i,\text{post}} - x_{i,\text{pre}}\) 2. Rank the absolute values \(|d_i|\) (excluding zeros) 3. Assign signs to ranks based on the sign of \(d_i\) 4. Test statistic: \(W^+ = \sum_{d_i > 0} \text{rank}(|d_i|)\)

Hypotheses (two-tailed): \(H_0: \tilde{d} = 0\) vs \(H_1: \tilde{d} \neq 0\)

# Paired comparison: pain score baseline vs. week 8 (Active arm only)
active_pain <- pain_data |> filter(arm == "Active")

# Parametric paired t-test for comparison
t_paired <- t.test(active_pain$pain_week8, active_pain$pain_base, paired = TRUE)

# Wilcoxon signed-rank test
w_signed <- wilcox.test(active_pain$pain_week8, active_pain$pain_base,
                        paired = TRUE, conf.int = TRUE)

cat("=== Paired Comparison: Pain Score (Active Arm) ===\n\n")
#> === Paired Comparison: Pain Score (Active Arm) ===
cat("Parametric paired t-test:\n")
#> Parametric paired t-test:
cat("  t =", round(t_paired$statistic, 3),
    ", p =", round(t_paired$p.value, 4), "\n")
#>   t = -4.929 , p = 0
cat("  95% CI:", round(t_paired$conf.int[1], 2), "to",
    round(t_paired$conf.int[2], 2), "\n\n")
#>   95% CI: -3.96 to -1.64
cat("Wilcoxon signed-rank test:\n")
#> Wilcoxon signed-rank test:
cat("  V =", w_signed$statistic,
    ", p =", round(w_signed$p.value, 4), "\n")
#>   V = 38 , p = 3e-04
cat("  95% CI (pseudomedian):", round(w_signed$conf.int[1], 2),
    "to", round(w_signed$conf.int[2], 2), "\n")
#>   95% CI (pseudomedian): -4.5 to -2

13.4 3. Mann–Whitney U Test (Wilcoxon Rank-Sum Test)

13.4.1 3.1 Concept

The Mann–Whitney U test (equivalent to the Wilcoxon rank-sum test) compares the distribution of outcomes between two independent groups. It tests whether one group tends to have higher values than the other.

Hypotheses: \(H_0\): The two populations are identically distributed \(H_1\): One population tends to produce larger values

The test statistic \(U\) counts the number of times an observation from group 1 exceeds an observation from group 2.

# Compare pain changes between Active and Placebo
mw_result <- wilcox.test(pain_chg ~ arm, data = pain_data,
                         conf.int = TRUE, conf.level = 0.95)

# For comparison, parametric test
t_indep <- t.test(pain_chg ~ arm, data = pain_data, var.equal = FALSE)

cat("=== Pain Change: Active vs. Placebo ===\n\n")
#> === Pain Change: Active vs. Placebo ===
cat("Medians:\n")
#> Medians:
pain_data |>
  group_by(arm) |>
  summarise(median = median(pain_chg), IQR = IQR(pain_chg)) |>
  print()
#> # A tibble: 2 × 3
#>   arm     median   IQR
#>   <chr>    <dbl> <dbl>
#> 1 Active    -3    3.5 
#> 2 Placebo   -0.5  2.75
cat("\nMann-Whitney U test:\n")
#> 
#> Mann-Whitney U test:
cat("  W =", mw_result$statistic,
    ", p =", round(mw_result$p.value, 4), "\n")
#>   W = 214.5 , p = 5e-04
cat("  Hodges-Lehmann estimate:", round(mw_result$estimate, 2), "\n")
#>   Hodges-Lehmann estimate: -3
cat("  95% CI:", round(mw_result$conf.int[1], 2), "to",
    round(mw_result$conf.int[2], 2), "\n\n")
#>   95% CI: -4 to -1
cat("Two-sample t-test (for comparison):\n")
#> Two-sample t-test (for comparison):
cat("  t =", round(t_indep$statistic, 3),
    ", p =", round(t_indep$p.value, 4), "\n")
#>   t = -3.477 , p = 0.001

13.4.2 3.2 The Probability of Superiority

A useful effect size for the Mann–Whitney test is the common language effect size (CLES), also called the probability of superiority:

\[P(\text{Active} < \text{Placebo}) = \frac{U}{n_1 \cdot n_2}\]

active_vals  <- pain_data |> filter(arm == "Active")  |> pull(pain_chg)
placebo_vals <- pain_data |> filter(arm == "Placebo") |> pull(pain_chg)

# CLES: P(Active < Placebo) for pain (lower = better)
u_stat <- mw_result$statistic
cles   <- as.numeric(u_stat) / (length(active_vals) * length(placebo_vals))

cat("Probability that Active patient has lower pain change than Placebo patient:\n")
#> Probability that Active patient has lower pain change than Placebo patient:
cat("  CLES =", round(cles, 3), "\n")
#>   CLES = 0.238
cat("  Interpretation: In", round(cles * 100, 1),
    "% of Active–Placebo pairs, the Active patient had more pain reduction\n")
#>   Interpretation: In 23.8 % of Active–Placebo pairs, the Active patient had more pain reduction
pain_data |>
  ggplot(aes(x = arm, y = pain_chg, fill = arm)) +
  geom_boxplot(alpha = 0.75, width = 0.5, outlier.shape = 21) +
  geom_jitter(width = 0.15, alpha = 0.5, size = 1.5) +
  scale_fill_manual(values = c("Active" = "#2C6E9E", "Placebo" = "#E07B39")) +
  annotate("text", x = 1.5, y = max(pain_data$pain_chg) * 0.95,
           label = paste0("Mann-Whitney p = ", round(mw_result$p.value, 3)),
           size = 3.5, fontface = "italic") +
  labs(
    x = "Treatment Arm",
    y = "Change in Pain Score",
    fill = NULL,
    title = "Pain Score Changes: Active vs. Placebo",
    subtitle = "Box shows median and IQR; points show individual values"
  ) +
  theme_bw(base_size = 12) +
  theme(legend.position = "none")

13.5 4. Kruskal–Wallis Test

13.5.1 4.1 Extension to Multiple Groups

The Kruskal–Wallis test extends the Mann–Whitney approach to three or more independent groups. It tests whether the population medians are equal.

\[H = \frac{12}{N(N+1)} \sum_{i=1}^{k} \frac{R_i^2}{n_i} - 3(N+1)\]

where \(R_i\) is the sum of ranks in group \(i\), \(n_i\) is the group size, and \(N\) is the total sample size. Under \(H_0\), \(H\) follows a \(\chi^2\) distribution with \(k-1\) df.

# Three-group comparison: dose-response study
set.seed(55)
dose_data <- tibble(
  subject_id = paste0("DPT-", sprintf("%03d", 1:75)),
  dose       = rep(c("Placebo", "Low Dose", "High Dose"), each = 25),
  response   = c(
    round(rbeta(25, 2.5, 2) * 10),        # Placebo
    round(rbeta(25, 2, 2.5) * 10),        # Low dose
    round(rbeta(25, 1.5, 3.5) * 10)       # High dose
  )
)

# Order factor
dose_data <- dose_data |>
  mutate(dose = factor(dose, levels = c("Placebo", "Low Dose", "High Dose")))

kw_result <- kruskal.test(response ~ dose, data = dose_data)

cat("Kruskal-Wallis Test: Response by Dose Group\n")
#> Kruskal-Wallis Test: Response by Dose Group
cat("  H =", round(kw_result$statistic, 3),
    ", df =", kw_result$parameter,
    ", p =", round(kw_result$p.value, 4), "\n\n")
#>   H = 15.83 , df = 2 , p = 4e-04
dose_data |>
  group_by(dose) |>
  summarise(n = n(), median = median(response), IQR = IQR(response)) |>
  gt() |>
  tab_header(title = "Summary Statistics by Dose Group")

13.6 5. Friedman Test: Repeated Measures

The Friedman test is the non-parametric equivalent of repeated-measures ANOVA. It is used when the same subjects are measured under \(k\) conditions.

# Repeated measures: pain scores at 3 timepoints (within subject)
set.seed(77)
n_subj <- 30
long_pain <- tibble(
  subject_id = rep(paste0("PT-", sprintf("%03d", 1:n_subj)), times = 3),
  timepoint  = rep(c("Baseline", "Week 4", "Week 8"), each = n_subj),
  pain_score = c(
    round(rbeta(n_subj, 3, 1.5) * 10),         # Baseline
    round(rbeta(n_subj, 2.5, 2) * 10),          # Week 4
    round(rbeta(n_subj, 1.5, 3.5) * 10)         # Week 8
  )
) |>
  mutate(timepoint = factor(timepoint,
                            levels = c("Baseline", "Week 4", "Week 8")))

# Wide format for Friedman
wide_pain <- long_pain |>
  pivot_wider(names_from = timepoint, values_from = pain_score)

friedman_result <- friedman.test(
  y = as.matrix(wide_pain[, c("Baseline", "Week 4", "Week 8")])
)

cat("Friedman Test: Pain Scores at 3 Timepoints\n")
#> Friedman Test: Pain Scores at 3 Timepoints
cat("  χ² =", round(friedman_result$statistic, 3),
    ", df =", friedman_result$parameter,
    ", p =", round(friedman_result$p.value, 4), "\n")
#>   χ² = 23.636 , df = 2 , p = 0
# Descriptive statistics
long_pain |>
  group_by(timepoint) |>
  summarise(
    n      = n(),
    Median = median(pain_score),
    IQR    = IQR(pain_score),
    Mean   = round(mean(pain_score), 1)
  ) |>
  gt() |>
  tab_header(title = "Pain Scores by Timepoint")

13.7 6. Spearman Rank Correlation

Spearman’s \(\rho\) measures the strength of a monotonic (not necessarily linear) relationship between two variables. It is the correlation of the ranked values:

\[r_s = 1 - \frac{6 \sum d_i^2}{n(n^2 - 1)}\]

where \(d_i\) is the difference in ranks for observation \(i\).

set.seed(101)
n_corr <- 60
cor_data <- tibble(
  age         = runif(n_corr, 40, 80),
  pain_score  = 2 + 0.05 * age + rnorm(n_corr, 0, 1.5),   # monotonic but noisy
  crp         = exp(0.03 * age + rnorm(n_corr, 0, 0.5))     # non-linear (log-normal)
)

r_pearson  <- cor(cor_data$age, cor_data$crp, method = "pearson")
r_spearman <- cor(cor_data$age, cor_data$crp, method = "spearman")
cor_test   <- cor.test(cor_data$age, cor_data$crp, method = "spearman")

cat("Pearson r:   ", round(r_pearson,  3), "\n")
#> Pearson r:    0.546
cat("Spearman rho:", round(r_spearman, 3),
    " (p =", round(cor_test$p.value, 4), ")\n")
#> Spearman rho: 0.608  (p = 0 )
cor_data |>
  ggplot(aes(x = age, y = crp)) +
  geom_point(colour = "#2C6E9E", alpha = 0.7, size = 2) +
  geom_smooth(method = "lm", colour = "red",   linetype = "dashed",
              se = FALSE, linewidth = 0.9) +
  geom_smooth(method = "loess", colour = "#E07B39",
              se = FALSE, linewidth = 0.9) +
  annotate("text", x = 42, y = max(cor_data$crp) * 0.95,
           label = paste0("Pearson r = ", round(r_pearson, 2),
                          "\nSpearman ρ = ", round(r_spearman, 2)),
           hjust = 0, size = 3.5) +
  labs(
    x = "Age (years)",
    y = "CRP (mg/L)",
    title = "Age vs. CRP: Spearman Captures the Monotonic Relationship",
    subtitle = "Red dashed = linear fit; orange = LOESS"
  ) +
  theme_bw(base_size = 12)

13.8 7. Permutation Tests

Permutation tests provide exact non-parametric p-values by computing the test statistic under all (or a random sample of) permutations of group labels.

set.seed(123)
obs_diff   <- mean(active_vals) - mean(placebo_vals)
all_values <- c(active_vals, placebo_vals)
n1         <- length(active_vals)
n_perm     <- 5000

perm_diffs <- replicate(n_perm, {
  shuffled <- sample(all_values)
  mean(shuffled[1:n1]) - mean(shuffled[(n1 + 1):length(all_values)])
})

perm_p <- mean(abs(perm_diffs) >= abs(obs_diff))

cat("Observed mean difference:", round(obs_diff, 3), "\n")
#> Observed mean difference: -2.567
cat("Permutation p-value:      ", round(perm_p, 4), "\n")
#> Permutation p-value:       2e-04
cat("Mann-Whitney p-value:     ", round(mw_result$p.value, 4), "\n")
#> Mann-Whitney p-value:      5e-04
tibble(perm_diff = perm_diffs) |>
  ggplot(aes(x = perm_diff)) +
  geom_histogram(bins = 50, fill = "#757575", colour = "white", alpha = 0.8) +
  geom_vline(xintercept =  obs_diff, colour = "red",
             linewidth = 1.1, linetype = "dashed") +
  geom_vline(xintercept = -obs_diff, colour = "red",
             linewidth = 1.1, linetype = "dashed") +
  annotate("text", x = obs_diff + 0.2, y = 400,
           label = paste0("Observed\n", round(obs_diff, 2)),
           colour = "red", hjust = 0, size = 3.5) +
  labs(
    x = "Permutation Mean Difference",
    y = "Count",
    title = "Permutation Distribution of Mean Difference",
    subtitle = paste0(n_perm, " permutations; p = ", round(perm_p, 4))
  ) +
  theme_bw(base_size = 12)

13.10 9. Practical Example: Patient-Reported Outcome Comparison

# PRO data: EQ-5D utility score (0-1, ordinal-like, bounded)
set.seed(88)
pro_data <- tibble(
  subject_id = paste0("SUBJ-", sprintf("%03d", 1:100)),
  arm        = rep(c("Active", "Placebo"), each = 50),
  eq5d_base  = round(rbeta(100, 3, 2), 2),
  eq5d_w12   = c(
    pmin(1, round(rbeta(50, 4, 1.5) + 0.05, 2)),  # Active: improvement

13.11 Summary

Non-parametric tests are indispensable tools in the clinical researcher’s toolkit:

  • Use them when data are ordinal, heavily skewed, or when sample sizes are too small to verify normality
  • Wilcoxon signed-rank replaces the paired t-test; Mann–Whitney U replaces the independent t-test
  • Kruskal–Wallis handles three or more independent groups; follow with pairwise Wilcoxon post-hoc tests
  • Friedman test handles repeated measures when normality fails
  • Spearman ρ captures monotonic (non-linear) associations; appropriate for ordinal variables
  • Permutation tests provide exact p-values with no distributional assumptions
  • Non-parametric tests lose some power relative to parametric tests when normality holds — but in clinical research with small or ordinal data, the extra robustness is worthwhile

Regulatory note: For PRO endpoints scored on ordinal scales (e.g., Likert scales, VAS), regulatory guidance often requires pre-specification of whether parametric or non-parametric analysis will be used in the SAP, with justification based on distributional assumptions.

# Non-Parametric Tests in Clinical Research

## Introduction

Non-parametric (distribution-free) tests make fewer assumptions about the underlying data distribution. They are particularly valuable in clinical research when:

- Sample sizes are small (n < 30) and normality cannot be verified
- Data are ordinal (e.g., pain scores, quality-of-life ratings)
- Distributions are heavily skewed or contain outliers
- Outcomes are censored counts or time-to-event with small samples

This chapter covers:

- **Wilcoxon signed-rank test**: Paired or one-sample comparisons
- **Mann–Whitney U test**: Two independent groups
- **Kruskal–Wallis test**: Multiple independent groups
- **Friedman test**: Repeated measures across groups
- **Spearman correlation**: Monotonic association between variables
- **Permutation tests**: Exact non-parametric inference
- **Choosing between parametric and non-parametric approaches**

## 2. Wilcoxon Signed-Rank Test

### 2.1 Concept and Hypotheses

The Wilcoxon signed-rank test evaluates whether the median of paired differences (or a one-sample median) equals a hypothesised value. It replaces the paired t-test when the normality assumption fails.

**Procedure**:
1. Compute pairwise differences $d_i = x_{i,\text{post}} - x_{i,\text{pre}}$
2. Rank the absolute values $|d_i|$ (excluding zeros)
3. Assign signs to ranks based on the sign of $d_i$
4. Test statistic: $W^+ = \sum_{d_i > 0} \text{rank}(|d_i|)$

**Hypotheses** (two-tailed): $H_0: \tilde{d} = 0$ vs $H_1: \tilde{d} \neq 0$

```{r}
#| label: signed-rank-test

# Paired comparison: pain score baseline vs. week 8 (Active arm only)
active_pain <- pain_data |> filter(arm == "Active")

# Parametric paired t-test for comparison
t_paired <- t.test(active_pain$pain_week8, active_pain$pain_base, paired = TRUE)

# Wilcoxon signed-rank test
w_signed <- wilcox.test(active_pain$pain_week8, active_pain$pain_base,
                        paired = TRUE, conf.int = TRUE)

cat("=== Paired Comparison: Pain Score (Active Arm) ===\n\n")
cat("Parametric paired t-test:\n")
cat("  t =", round(t_paired$statistic, 3),
    ", p =", round(t_paired$p.value, 4), "\n")
cat("  95% CI:", round(t_paired$conf.int[1], 2), "to",
    round(t_paired$conf.int[2], 2), "\n\n")

cat("Wilcoxon signed-rank test:\n")
cat("  V =", w_signed$statistic,
    ", p =", round(w_signed$p.value, 4), "\n")
cat("  95% CI (pseudomedian):", round(w_signed$conf.int[1], 2),
    "to", round(w_signed$conf.int[2], 2), "\n")
```

## 3. Mann–Whitney U Test (Wilcoxon Rank-Sum Test)

### 3.1 Concept

The Mann–Whitney U test (equivalent to the Wilcoxon rank-sum test) compares the distribution of outcomes between two independent groups. It tests whether one group tends to have higher values than the other.

**Hypotheses**: $H_0$: The two populations are identically distributed
$H_1$: One population tends to produce larger values

The test statistic $U$ counts the number of times an observation from group 1 exceeds an observation from group 2.

```{r}
#| label: mann-whitney

# Compare pain changes between Active and Placebo
mw_result <- wilcox.test(pain_chg ~ arm, data = pain_data,
                         conf.int = TRUE, conf.level = 0.95)

# For comparison, parametric test
t_indep <- t.test(pain_chg ~ arm, data = pain_data, var.equal = FALSE)

cat("=== Pain Change: Active vs. Placebo ===\n\n")
cat("Medians:\n")
pain_data |>
  group_by(arm) |>
  summarise(median = median(pain_chg), IQR = IQR(pain_chg)) |>
  print()

cat("\nMann-Whitney U test:\n")
cat("  W =", mw_result$statistic,
    ", p =", round(mw_result$p.value, 4), "\n")
cat("  Hodges-Lehmann estimate:", round(mw_result$estimate, 2), "\n")
cat("  95% CI:", round(mw_result$conf.int[1], 2), "to",
    round(mw_result$conf.int[2], 2), "\n\n")

cat("Two-sample t-test (for comparison):\n")
cat("  t =", round(t_indep$statistic, 3),
    ", p =", round(t_indep$p.value, 4), "\n")
```

### 3.2 The Probability of Superiority

A useful effect size for the Mann–Whitney test is the **common language effect size** (CLES), also called the probability of superiority:

$$P(\text{Active} < \text{Placebo}) = \frac{U}{n_1 \cdot n_2}$$

```{r}
#| label: cles

active_vals  <- pain_data |> filter(arm == "Active")  |> pull(pain_chg)
placebo_vals <- pain_data |> filter(arm == "Placebo") |> pull(pain_chg)

# CLES: P(Active < Placebo) for pain (lower = better)
u_stat <- mw_result$statistic
cles   <- as.numeric(u_stat) / (length(active_vals) * length(placebo_vals))

cat("Probability that Active patient has lower pain change than Placebo patient:\n")
cat("  CLES =", round(cles, 3), "\n")
cat("  Interpretation: In", round(cles * 100, 1),
    "% of Active–Placebo pairs, the Active patient had more pain reduction\n")
```

## 4. Kruskal–Wallis Test

### 4.1 Extension to Multiple Groups

The Kruskal–Wallis test extends the Mann–Whitney approach to three or more independent groups. It tests whether the population medians are equal.

$$H = \frac{12}{N(N+1)} \sum_{i=1}^{k} \frac{R_i^2}{n_i} - 3(N+1)$$

where $R_i$ is the sum of ranks in group $i$, $n_i$ is the group size, and $N$ is the total sample size. Under $H_0$, $H$ follows a $\chi^2$ distribution with $k-1$ df.

```{r}
#| label: kruskal-wallis

# Three-group comparison: dose-response study
set.seed(55)
dose_data <- tibble(
  subject_id = paste0("DPT-", sprintf("%03d", 1:75)),
  dose       = rep(c("Placebo", "Low Dose", "High Dose"), each = 25),
  response   = c(
    round(rbeta(25, 2.5, 2) * 10),        # Placebo
    round(rbeta(25, 2, 2.5) * 10),        # Low dose
    round(rbeta(25, 1.5, 3.5) * 10)       # High dose
  )
)

# Order factor
dose_data <- dose_data |>
  mutate(dose = factor(dose, levels = c("Placebo", "Low Dose", "High Dose")))

kw_result <- kruskal.test(response ~ dose, data = dose_data)

cat("Kruskal-Wallis Test: Response by Dose Group\n")
cat("  H =", round(kw_result$statistic, 3),
    ", df =", kw_result$parameter,
    ", p =", round(kw_result$p.value, 4), "\n\n")

dose_data |>
  group_by(dose) |>
  summarise(n = n(), median = median(response), IQR = IQR(response)) |>
  gt() |>
  tab_header(title = "Summary Statistics by Dose Group")
```

## 5. Friedman Test: Repeated Measures

The Friedman test is the non-parametric equivalent of repeated-measures ANOVA. It is used when the same subjects are measured under $k$ conditions.

```{r}

# Repeated measures: pain scores at 3 timepoints (within subject)
set.seed(77)
n_subj <- 30
long_pain <- tibble(
  subject_id = rep(paste0("PT-", sprintf("%03d", 1:n_subj)), times = 3),
```

## 6. Spearman Rank Correlation

Spearman's $\rho$ measures the strength of a monotonic (not necessarily linear) relationship between two variables. It is the correlation of the ranked values:

$$r_s = 1 - \frac{6 \sum d_i^2}{n(n^2 - 1)}$$

where $d_i$ is the difference in ranks for observation $i$.

```{r}
#| label: spearman-correlation
#| fig-cap: "Spearman vs. Pearson correlation: Spearman captures monotonic but non-linear relationships"

set.seed(101)
n_corr <- 60
cor_data <- tibble(
  age         = runif(n_corr, 40, 80),
  pain_score  = 2 + 0.05 * age + rnorm(n_corr, 0, 1.5),   # monotonic but noisy
  crp         = exp(0.03 * age + rnorm(n_corr, 0, 0.5))     # non-linear (log-normal)
)

r_pearson  <- cor(cor_data$age, cor_data$crp, method = "pearson")
r_spearman <- cor(cor_data$age, cor_data$crp, method = "spearman")
cor_test   <- cor.test(cor_data$age, cor_data$crp, method = "spearman")

cat("Pearson r:   ", round(r_pearson,  3), "\n")
cat("Spearman rho:", round(r_spearman, 3),
    " (p =", round(cor_test$p.value, 4), ")\n")

cor_data |>
  ggplot(aes(x = age, y = crp)) +
  geom_point(colour = "#2C6E9E", alpha = 0.7, size = 2) +
  geom_smooth(method = "lm", colour = "red",   linetype = "dashed",
              se = FALSE, linewidth = 0.9) +
  geom_smooth(method = "loess", colour = "#E07B39",
              se = FALSE, linewidth = 0.9) +
  annotate("text", x = 42, y = max(cor_data$crp) * 0.95,
           label = paste0("Pearson r = ", round(r_pearson, 2),
                          "\nSpearman ρ = ", round(r_spearman, 2)),
           hjust = 0, size = 3.5) +
  labs(
    x = "Age (years)",
    y = "CRP (mg/L)",
    title = "Age vs. CRP: Spearman Captures the Monotonic Relationship",
    subtitle = "Red dashed = linear fit; orange = LOESS"
  ) +
```

## 7. Permutation Tests

Permutation tests provide exact non-parametric p-values by computing the test statistic under all (or a random sample of) permutations of group labels.

```{r}
#| label: permutation-test
#| fig-cap: "Permutation distribution of mean difference; observed value shown in red"

set.seed(123)
obs_diff   <- mean(active_vals) - mean(placebo_vals)
all_values <- c(active_vals, placebo_vals)
n1         <- length(active_vals)
n_perm     <- 5000

perm_diffs <- replicate(n_perm, {
  shuffled <- sample(all_values)
  mean(shuffled[1:n1]) - mean(shuffled[(n1 + 1):length(all_values)])
})

perm_p <- mean(abs(perm_diffs) >= abs(obs_diff))

cat("Observed mean difference:", round(obs_diff, 3), "\n")
cat("Permutation p-value:      ", round(perm_p, 4), "\n")
cat("Mann-Whitney p-value:     ", round(mw_result$p.value, 4), "\n")

tibble(perm_diff = perm_diffs) |>
  ggplot(aes(x = perm_diff)) +
  geom_histogram(bins = 50, fill = "#757575", colour = "white", alpha = 0.8) +
  geom_vline(xintercept =  obs_diff, colour = "red",
             linewidth = 1.1, linetype = "dashed") +
  geom_vline(xintercept = -obs_diff, colour = "red",
             linewidth = 1.1, linetype = "dashed") +
  annotate("text", x = obs_diff + 0.2, y = 400,
           label = paste0("Observed\n", round(obs_diff, 2)),
           colour = "red", hjust = 0, size = 3.5) +
  labs(
    x = "Permutation Mean Difference",
    y = "Count",
    title = "Permutation Distribution of Mean Difference",
    subtitle = paste0(n_perm, " permutations; p = ", round(perm_p, 4))
  ) +
```

## Summary

Non-parametric tests are indispensable tools in the clinical researcher's toolkit:

- **Use them** when data are ordinal, heavily skewed, or when sample sizes are too small to verify normality
- **Wilcoxon signed-rank** replaces the paired t-test; **Mann–Whitney U** replaces the independent t-test
- **Kruskal–Wallis** handles three or more independent groups; follow with pairwise Wilcoxon post-hoc tests
- **Friedman test** handles repeated measures when normality fails
- **Spearman ρ** captures monotonic (non-linear) associations; appropriate for ordinal variables
- **Permutation tests** provide exact p-values with no distributional assumptions
- Non-parametric tests lose some power relative to parametric tests when normality holds — but in clinical research with small or ordinal data, the extra robustness is worthwhile

> **Regulatory note**: For PRO endpoints scored on ordinal scales (e.g., Likert scales, VAS), regulatory guidance often requires pre-specification of whether parametric or non-parametric analysis will be used in the SAP, with justification based on distributional assumptions.
