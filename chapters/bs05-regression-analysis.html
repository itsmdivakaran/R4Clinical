14  Regression Analysis in Clinical Trials

Regression models are among the most widely used statistical tools in clinical research. They allow us to quantify relationships between a response variable and one or more predictors, adjust for confounders, and make predictions. This chapter covers linear regression, logistic regression, and Poisson regression, all illustrated with the pharmaversesdtm clinical datasets.

14.1 Setup and Data Loading

library(dplyr)
library(ggplot2)

14.2 1. Simple Linear Regression

We predict systolic blood pressure (SBP) from subject age. SBP is stored in the vs dataset where VSTESTCD == "SYSBP".

14.2.1 1.1 Preparing the SBP Dataset

library(dplyr)

# Extract baseline SBP values (VISIT == "BASELINE" or first available)
sbp_data <- vs |>
  filter(VSTESTCD == "SYSBP", !is.na(VSSTRESN)) |>
  group_by(USUBJID) |>
  slice_min(order_by = VISITNUM, n = 1, with_ties = FALSE) |>
  ungroup() |>
  select(USUBJID, SBP = VSSTRESN)

# Join to demographics
analysis_df <- dm_clean |>
  inner_join(sbp_data, by = "USUBJID") |>
  filter(!is.na(AGE), !is.na(SBP)) |>
  mutate(SEX = factor(SEX),
         RACE = factor(RACE))

14.4 3. Regression Diagnostics

Before drawing conclusions, we must verify that the four key assumptions of linear regression hold:

  1. Linearity – the relationship is approximately linear.
  2. Independence – residuals are not correlated.
  3. Homoscedasticity – residuals have constant variance.
  4. Normality – residuals are approximately normally distributed.

14.4.1 3.1 Residual Plots

par(mfrow = c(2, 2))
plot(model_multi, which = 1:4)
par(mfrow = c(1, 1))
  • Residuals vs Fitted: checks linearity and homoscedasticity — a random scatter around zero is ideal.
  • Normal Q-Q: checks normality — points should lie close to the diagonal.
  • Scale-Location: another view of homoscedasticity.
  • Cook’s Distance / Residuals vs Leverage: identifies influential observations.

14.4.2 3.2 Cook’s Distance — Identifying Influential Points

library(dplyr)

cooks_d <- cooks.distance(model_multi)
influential_threshold <- 4 / nrow(analysis_df)

14.5 4. Model Selection

14.5.1 4.1 AIC Comparison

AIC (Akaike Information Criterion) penalises model complexity. Lower AIC indicates a better balance between fit and parsimony.

models <- list(
  "Age only"       = lm(SBP ~ AGE, data = analysis_df),
  "Age + Sex"      = lm(SBP ~ AGE + SEX, data = analysis_df),
  "Age + Sex + Race" = lm(SBP ~ AGE + SEX + RACE, data = analysis_df),
  "Full model"     = model_multi
)

aic_tbl <- data.frame(
  Model = names(models),
  AIC   = sapply(models, AIC),
  BIC   = sapply(models, BIC),
  R2    = sapply(models, \(m) summary(m)$r.squared)
)
aic_tbl <- aic_tbl[order(aic_tbl$AIC), ]
aic_tbl[, -1] <- round(aic_tbl[, -1], 3)
print(aic_tbl)
#>                             Model      AIC      BIC    R2

14.8 7. Model Evaluation

14.8.1 7.1 Confusion Matrix

We use a 0.5 probability threshold to classify predictions.

pred_probs <- predict(model_logit, newdata = logit_df, type = "response")
pred_class <- factor(if_else(pred_probs >= 0.5, "AE", "No AE"),
                     levels = c("No AE", "AE"))
actual     <- factor(as.character(logit_df$ANY_AE), levels = c("No AE", "AE"))

conf_mat <- table(
  Predicted = pred_class,
  Actual    = actual
)
print(conf_mat)
#>          Actual
#> Predicted No AE  AE
#>     No AE     0   0
#>     AE       29 225
# Derived metrics (with safe fallbacks for zero cells)
tp <- conf_mat["AE", "AE"]
tn <- conf_mat["No AE", "No AE"]
fp <- conf_mat["AE", "No AE"]
fn <- conf_mat["No AE", "AE"]

sensitivity <- if ((tp + fn) > 0) tp / (tp + fn) else NA_real_
specificity  <- if ((tn + fp) > 0) tn / (tn + fp) else NA_real_
accuracy     <- (tp + tn) / sum(conf_mat)

cat(sprintf("\nSensitivity : %.3f\n", sensitivity))
#> 
#> Sensitivity : 1.000
cat(sprintf("Specificity : %.3f\n", specificity))
#> Specificity : 0.000
cat(sprintf("Accuracy    : %.3f\n", accuracy))
#> Accuracy    : 0.886

14.8.2 7.2 ROC Curve Concept

The ROC curve plots sensitivity vs (1 - specificity) across all classification thresholds. Here we build it manually without additional packages.

library(dplyr)
library(ggplot2)

roc_df <- data.frame(
  prob   = pred_probs,
  actual = logit_df$ANY_AE == "AE"
) |>
  arrange(desc(prob))

thresholds <- sort(unique(roc_df$prob), decreasing = TRUE)

roc_points <- lapply(thresholds, function(thr) {
  pred_pos <- roc_df$prob >= thr
  tp_i  <- sum( pred_pos &  roc_df$actual)
  fp_i  <- sum( pred_pos & !roc_df$actual)
  fn_i  <- sum(!pred_pos &  roc_df$actual)
  tn_i  <- sum(!pred_pos & !roc_df$actual)
  data.frame(
    TPR = tp_i / (tp_i + fn_i),
    FPR = fp_i / (fp_i + tn_i)
  )
}) |>
  dplyr::bind_rows()

# Add (0,0) and (1,1)
roc_curve <- rbind(data.frame(TPR = 0, FPR = 0),
                    roc_points,
                    data.frame(TPR = 1, FPR = 1))

ggplot(roc_curve, aes(x = FPR, y = TPR)) +
  geom_line(colour = "#2166ac", linewidth = 1.1) +
  geom_abline(slope = 1, intercept = 0, linetype = "dashed",
              colour = "grey60") +
  labs(title = "ROC Curve — Logistic Regression (AE Occurrence)",
       x = "1 - Specificity (False Positive Rate)",

# Regression Analysis in Clinical Trials {#sec-regression}

Regression models are among the most widely used statistical tools in clinical research. They allow us to quantify relationships between a response variable and one or more predictors, adjust for confounders, and make predictions. This chapter covers linear regression, logistic regression, and Poisson regression, all illustrated with the `pharmaversesdtm` clinical datasets.

## 3. Regression Diagnostics {#sec-diagnostics}

Before drawing conclusions, we must verify that the four key assumptions of linear regression hold:

1. **Linearity** – the relationship is approximately linear.
2. **Independence** – residuals are not correlated.
3. **Homoscedasticity** – residuals have constant variance.
4. **Normality** – residuals are approximately normally distributed.

### 3.1 Residual Plots

```{r diagnostic-plots, fig.height=8, fig.width=9}
par(mfrow = c(2, 2))
plot(model_multi, which = 1:4)
par(mfrow = c(1, 1))
```

- **Residuals vs Fitted**: checks linearity and homoscedasticity — a random scatter around zero is ideal.
- **Normal Q-Q**: checks normality — points should lie close to the diagonal.
- **Scale-Location**: another view of homoscedasticity.
- **Cook's Distance / Residuals vs Leverage**: identifies influential observations.

### 3.2 Cook's Distance — Identifying Influential Points

```{r cooks}
library(dplyr)

cooks_d <- cooks.distance(model_multi)
influential_threshold <- 4 / nrow(analysis_df)

influential_df <- data.frame(
  USUBJID = analysis_df$USUBJID,
  CooksD   = cooks_d

  filter(CooksD > influential_threshold) |>
  arrange(desc(CooksD))
```

## 7. Model Evaluation {#sec-evaluation}

### 7.1 Confusion Matrix

We use a 0.5 probability threshold to classify predictions.

```{r confusion-matrix}
pred_probs <- predict(model_logit, newdata = logit_df, type = "response")
pred_class <- factor(if_else(pred_probs >= 0.5, "AE", "No AE"),
                     levels = c("No AE", "AE"))
actual     <- factor(as.character(logit_df$ANY_AE), levels = c("No AE", "AE"))

conf_mat <- table(
  Predicted = pred_class,
  Actual    = actual
)
print(conf_mat)

# Derived metrics (with safe fallbacks for zero cells)
tp <- conf_mat["AE", "AE"]
tn <- conf_mat["No AE", "No AE"]
fp <- conf_mat["AE", "No AE"]
fn <- conf_mat["No AE", "AE"]

sensitivity <- if ((tp + fn) > 0) tp / (tp + fn) else NA_real_
specificity  <- if ((tn + fp) > 0) tn / (tn + fp) else NA_real_
accuracy     <- (tp + tn) / sum(conf_mat)

cat(sprintf("\nSensitivity : %.3f\n", sensitivity))
cat(sprintf("Specificity : %.3f\n", specificity))
cat(sprintf("Accuracy    : %.3f\n", accuracy))
```

### 7.2 ROC Curve Concept

The ROC curve plots sensitivity vs (1 - specificity) across all classification thresholds. Here we build it manually without additional packages.

```{r roc-manual}
library(dplyr)
library(ggplot2)

roc_df <- data.frame(
  prob   = pred_probs,
  actual = logit_df$ANY_AE == "AE"
) |>
  arrange(desc(prob))

thresholds <- sort(unique(roc_df$prob), decreasing = TRUE)

roc_points <- lapply(thresholds, function(thr) {
  pred_pos <- roc_df$prob >= thr
  tp_i  <- sum( pred_pos &  roc_df$actual)
  fp_i  <- sum( pred_pos & !roc_df$actual)
  fn_i  <- sum(!pred_pos &  roc_df$actual)
  tn_i  <- sum(!pred_pos & !roc_df$actual)
  data.frame(
    TPR = tp_i / (tp_i + fn_i),
    FPR = fp_i / (fp_i + tn_i)
  )
}) |>
  dplyr::bind_rows()

# Add (0,0) and (1,1)
roc_curve <- rbind(data.frame(TPR = 0, FPR = 0),
                    roc_points,
                    data.frame(TPR = 1, FPR = 1))

ggplot(roc_curve, aes(x = FPR, y = TPR)) +
  geom_line(colour = "#2166ac", linewidth = 1.1) +
  geom_abline(slope = 1, intercept = 0, linetype = "dashed",
              colour = "grey60") +
  labs(title = "ROC Curve — Logistic Regression (AE Occurrence)",
       x = "1 - Specificity (False Positive Rate)",
       y = "Sensitivity (True Positive Rate)") +
  coord_equal() +
  theme_bw(base_size = 13)
```

1. Fit a simple linear regression of diastolic blood pressure (`VSTESTCD == "DIABP"`) on AGE. Compare the slope to the SBP result.

2. Extend the multiple linear regression for SBP by including a RACE × SEX interaction term. Does the interaction improve model fit (use `anova()`)? Interpret any significant interaction.

3. Examine the Cook's distance plot for the multiple regression model. Remove the most influential observation and refit the model. How do the coefficients change?

4. Using `step()`, perform forward stepwise selection starting from an intercept-only model for SBP. Do you arrive at the same model as backward selection?

5. Fit a logistic regression predicting serious AEs (`AESER == "Y"` in the `ae` dataset) from age and ARM. Compute and interpret the odds ratio for the Xanomeline High Dose arm relative to Placebo.

6. The Poisson dispersion ratio you computed may exceed 1. Refit the AE-count model using `glm.nb()` from the `MASS` package (call it as `MASS::glm.nb()`). Compare AIC values to decide which model is preferable.

7. Add `WEIGHTBL` (baseline weight) as a covariate to the SBP multiple regression if that variable is available in `dm`. Interpret its coefficient.

8. Construct a calibration plot for the logistic regression model: bin subjects by predicted probability (e.g., deciles), compute the observed proportion of AEs in each bin, and plot observed vs predicted. Does the model appear well calibrated?
